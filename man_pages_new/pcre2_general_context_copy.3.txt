NAME

PCRE2 -
Perl-compatible regular expressions (revised API)

SYNOPSIS

#include

pcre2_general_context
*pcre2_general_context_copy( 
pcre2_general_context *gcontext);

DESCRIPTION

This function
makes a new copy of a general context, using the memory
allocation functions in the context, if set, to get the
necessary memory. Otherwise malloc() is used. The
result is NULL if the memory cannot be obtained.

There is a
complete description of the PCRE2 native API in the
pcre2api page and a description of the POSIX API in
the pcre2posix page.
