NAME

gh-codespace-rebuild
- Rebuild a codespace

SYNOPSIS

gh codespace
rebuild [flags]

DESCRIPTION

Rebuilding
recreates your codespace. Your code and any current changes
will be preserved. Your codespace will be rebuilt using your
working directory&rsquo;s dev container. A full rebuild also
removes cached Docker images.

OPTIONS

-c,
--codespace 

Name of the codespace

--full

perform a full rebuild

-R, --repo

Filter codespace selection by
repository name (user/repo)

--repo-owner

Filter codespace selection by
repository owner (username or org)

SEE ALSO

gh-codespace(1)
