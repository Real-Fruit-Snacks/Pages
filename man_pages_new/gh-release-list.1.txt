NAME

gh-release-list
- List releases in a repository

SYNOPSIS

gh release
list [flags]

OPTIONS

--exclude-drafts

Exclude draft releases

--exclude-pre-releases

Exclude pre-releases

-q, --jq

Filter JSON output using a jq
expression

--json

Output JSON with the specified
fields

-L, --limit
 (default 30)

Maximum number of items to
fetch

-O, --order
 (default "desc")

Order of releases returned:
{asc|desc}

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

SEE ALSO

gh-release(1)
