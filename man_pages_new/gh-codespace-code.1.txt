NAME

gh-codespace-code
- Open a codespace in Visual Studio Code

SYNOPSIS

gh codespace
code [flags]

OPTIONS

-c,
--codespace 

Name of the codespace

--insiders

Use the insiders version of
Visual Studio Code

-R, --repo

Filter codespace selection by
repository name (user/repo)

--repo-owner

Filter codespace selection by
repository owner (username or org)

-w, --web

Use the web version of Visual
Studio Code

SEE ALSO

gh-codespace(1)
