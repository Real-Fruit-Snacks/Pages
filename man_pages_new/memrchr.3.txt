NAME

memrchr - find
last occurrence of a character within another memory
block

SYNOPSIS

#include

void *memrchr(const void *v, int c,
size_t size);

DESCRIPTION

memrchr
finds the last occurrence of character c within
memory block v, of length size.

RETURN VALUE

memrchr
returns a pointer to the the match, if it finds any, or a
null pointer if it doesn&rsquo;t.
