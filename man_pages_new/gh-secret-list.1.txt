NAME

gh-secret-list
- List secrets

SYNOPSIS

gh secret
list [flags]

DESCRIPTION

List secrets on
one of the following levels: - repository (default):
available to GitHub Actions runs or Dependabot in a
repository - environment: available to GitHub Actions runs
for a deployment environment in a repository - organization:
available to GitHub Actions runs, Dependabot, or Codespaces
within an organization - user: available to Codespaces for
your user

OPTIONS

-a,
--app 

List secrets for a specific
application: {actions|codespaces|dependabot}

-e, --env

List secrets for an
environment

-q, --jq

Filter JSON output using a jq
expression

--json

Output JSON with the specified
fields

-o, --org

List secrets for an
organization

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

-u, --user

List a secret for your user

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format
