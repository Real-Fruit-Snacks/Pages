NAME

fc-conflist -
list the configuration files processed by Fontconfig

SYNOPSIS

fc-conflist
[ -Vh ]

[
--version ] [ --help ]

DESCRIPTION

fc-conflist
prints an annotated list of all the configuration files
processed by Fontconfig.

The output is a
&rsquo;-&rsquo; or &rsquo;+&rsquo; depending on whether the
file is ignored or processed, a space, the file&rsquo;s
path, a colon and space, and the description from the file
or &rsquo;No description&rsquo; if none is present.

The order of
files looks like how fontconfig actually processes them
except one contains  element. In that case,
it will be shown after processing all the sub directories
where is targeted by .

OPTIONS

This program
follows the usual GNU command line syntax, with long options
starting with two dashes (&rsquo;-&rsquo;). A summary of
options is included below.

-V

Show version of the program and exit.

-h

Show summary of options.

RETURN CODES

fc-conflist
returns error code 0 for successful parsing, or 1 if any
errors occurred or if at least one font face could not be
opened.
