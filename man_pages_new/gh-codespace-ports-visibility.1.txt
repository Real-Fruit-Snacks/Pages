NAME

gh-codespace-ports-visibility
- Change the visibility of the forwarded port

SYNOPSIS

gh codespace
ports visibility :{public|private|org}...
[flags]

OPTIONS INHERITED FROM PARENT COMMANDS

-c,
--codespace 

Name of the codespace

-R, --repo

Filter codespace selection by
repository name (user/repo)

--repo-owner

Filter codespace selection by
repository owner (username or org)

EXAMPLE

gh codespace
ports visibility 80:org 3000:private 8000:public

SEE ALSO

gh-codespace-ports(1)
