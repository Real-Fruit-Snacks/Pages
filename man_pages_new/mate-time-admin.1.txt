NAME

mate-time-admin
- set date and time

SYNOPSIS

mate-time-admin
[OPTIONS...]

DESCRIPTION

This executable
is part of the package &rsquo;mate-control-center&rsquo;:
The MATE Control Center.

Use
mate-time-admin , the MATE Time Admin Tool, to set
date and time of the local system.
