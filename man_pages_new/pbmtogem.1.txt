NAME

pbmtogem -
convert a PBM image into a GEM .img file

SYNOPSIS

pbmtogem
[pbmfile]

DESCRIPTION

This program is
part of Netpbm(1).

pbmtogem
reads a PBM image as input and produces a compressed GEM
.img file as output.

OPTIONS

There are no
command line options defined specifically for
pbmtogem, but it recognizes the options common to all
programs based on libnetpbm (See
 Common Options
.)

LIMITATIONS

pbmtogem
cannot compress repeated lines.
