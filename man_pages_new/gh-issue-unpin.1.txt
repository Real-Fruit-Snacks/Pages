NAME

gh-issue-unpin
- Unpin a issue

SYNOPSIS

gh issue
unpin { | } [flags]

DESCRIPTION

Unpin an issue
from a repository.

The issue can
be specified by issue number or URL.

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

EXAMPLE

# Unpin issue
from the current repository 
$ gh issue unpin 23

# Unpin issue
by URL 
$ gh issue unpin https://github.com/owner/repo/issues/23

# Unpin an
issue from specific repository 
$ gh issue unpin 23 --repo owner/repo
