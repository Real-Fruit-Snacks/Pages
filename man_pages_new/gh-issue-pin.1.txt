NAME

gh-issue-pin -
Pin a issue

SYNOPSIS

gh issue pin
{ | } [flags]

DESCRIPTION

Pin an issue to
a repository.

The issue can
be specified by issue number or URL.

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

EXAMPLE

# Pin an issue
to the current repository 
$ gh issue pin 23

# Pin an issue
by URL 
$ gh issue pin https://github.com/owner/repo/issues/23

# Pin an issue
to specific repository 
$ gh issue pin 23 --repo owner/repo
