NAME

setenforce -
modify the mode SELinux is running in

SYNOPSIS

setenforce
[Enforcing|Permissive|1|0]

DESCRIPTION

Use
Enforcing or 1 to put SELinux in enforcing
mode. 
Use Permissive or 0 to put SELinux in
permissive mode.

If SELinux is
disabled and you want to enable it, or SELinux is enabled
and you want to disable it, please see
selinux(8).
