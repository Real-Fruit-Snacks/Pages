NAME

gh-label-edit -
Edit a label

SYNOPSIS

gh label
edit  [flags]

DESCRIPTION

Update a label
on GitHub.

A label can be
renamed using the --name flag.

The label color
needs to be 6 character hex value.

OPTIONS

-c,
--color 

Color of the label

-d, --description

Description of the label

-n, --name

New name of the label

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

EXAMPLE

# update the
color of the bug label 
$ gh label edit bug --color FF0000

# rename and
edit the description of the bug label 
$ gh label edit bug --name big-bug --description
"Bigger than normal bug"
