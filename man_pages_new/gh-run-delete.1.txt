NAME

gh-run-delete -
Delete a workflow run

SYNOPSIS

gh run
delete [] [flags]

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

EXAMPLE

# Interactively
select a run to delete 
$ gh run delete

# Delete a
specific run 
$ gh run delete 12345

gh-run(1)
