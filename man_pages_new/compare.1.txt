NAME

compare -
compare an image to a reconstructed image.

SYNOPSIS

compare
file [ file ... ]

DESCRIPTION

Compare
- compare an image to a reconstructed image.

COMPARE OPTIONS

-authenticate

-colorspace

the type of colorspace

-compress

the type of image
compression

-debug

enable debug printout

-define
{=},...

options

-density
x

resolution in pixels of the image

-interlace

the type of interlacing
scheme

-limit 

Area, Disk, File, Map, or
Memory resource limit

-log

Specify format for debug
log

-metric type

Choose from MAE, MSE, PSE,
PSNR, or RMSE

-profile

add ICM, IPTC, or generic
profile to image

-quality

JPEG/MIFF/PNG compression
level

-sample

scale image using pixel
sampling

-size
x{+offset}

width and height of the
image

-version

print ImageMagick version
string

-virtual-pixel

specify contents of
"virtual pixels"
