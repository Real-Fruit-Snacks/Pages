FTPUSERS(5) BSD File Formats Manual FTPUSERS(5)

NAME

ftpusers &mdash; file
which lists users who are not allowed to use ftp

DESCRIPTION

/etc/ftpusers is used by
ftpd(8); the file contains a list of users who are not
allowed to use the ftp command. For security reasons at
least users like &rsquo;&rsquo;root&rsquo;&rsquo;,
&rsquo;&rsquo;bin&rsquo;&rsquo;,
&rsquo;&rsquo;uucp&rsquo;&rsquo; and
&rsquo;&rsquo;news&rsquo;&rsquo; should be listed in this
file. Blank lines and lines beginning with &rsquo;#&rsquo;
are ignored.

Note: a lines
with &rsquo;#&rsquo; in the middle is not a
comment. Don&rsquo;t put &rsquo;#&rsquo; after a name to
comment it; use another line, or things will silently fail
on you.

EXAMPLES

/etc/ftpusers might
contain the following entries:

# 
# /etc/ftpusers 
# 
root 
uucp 
news

FILES

/etc/ftpusers

SEE ALSO

ftp(1), ftpd(8)
