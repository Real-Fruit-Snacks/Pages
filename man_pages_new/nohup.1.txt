NAME

nohup - run a
command immune to hangups, with output to a non-tty

SYNOPSIS

nohup
COMMAND [ARG]... 
nohup OPTION

DESCRIPTION

Run COMMAND,
ignoring hangup signals.

--help

display this help and exit

--version

output version information and
exit

If standard
input is a terminal, redirect it from an unreadable file. If
standard output is a terminal, append output to
&rsquo;nohup.out&rsquo; if possible,
&rsquo;$HOME/nohup.out&rsquo; otherwise. If standard error
is a terminal, redirect it to standard output. To save
output to FILE, use &rsquo;nohup COMMAND >
FILE&rsquo;.

NOTE: your
shell may have its own version of nohup, which usually
supersedes the version described here. Please refer to your
shell&rsquo;s documentation for details about the options it
supports.

Exit
status:

125

if the nohup command itself fails

126

if COMMAND is found but cannot be invoked

127

if COMMAND cannot be found

-

the exit status of COMMAND otherwise
