NAME

pbmtosunicon -
convert a PBM image into a Sun icon

SYNOPSIS

pbmtosunicon
[pbmfile]

DESCRIPTION

This program is
part of Netpbm(1).

pbmtosunicon
reads a PBM image as input and produces a Sun icon as
output.

OPTIONS

There are no
command line options defined specifically for
pbmtosunicon, but it recognizes the options common to
all programs based on libnetpbm (See
 Common Options
.)
