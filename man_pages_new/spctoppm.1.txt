NAME

spctoppm -
convert an Atari compressed Spectrum file to a PPM

SYNOPSIS

spctoppm

[spcfile]

DESCRIPTION

This program is
part of Netpbm(1).

spctoppm
reads an Atari compressed Spectrum file as input and
produces a PPM image as output.

OPTIONS

There are no
command line options defined specifically for
spctoppm, but it recognizes the options common to all
programs based on libnetpbm (See
 Common Options
.)

sputoppm(1),
