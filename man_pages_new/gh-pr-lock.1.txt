NAME

gh-pr-lock -
Lock pull request conversation

SYNOPSIS

gh pr lock
{ | } [flags]

OPTIONS

-r,
--reason 

Optional reason for locking
conversation (off_topic, resolved, spam, too_heated).

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

SEE ALSO

gh-pr(1)
