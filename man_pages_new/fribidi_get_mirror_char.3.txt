NAME

fribidi_get_mirror_char
- get mirrored character

SYNOPSIS

#include

fribidi_boolean
fribidi_get_mirror_char 
(

FriBidiChar ch,

FriBidiChar *mirrored_ch

);

PARAMETERS

FriBidiChar
ch

Input character.

FriBidiChar
*mirrored_ch

Output mirrored character.

DESCRIPTION

This function
finds the mirrored equivalent of a character as defined in
the file BidiMirroring.txt of the Unicode Character Database
available at
http://www.unicode.org/Public/UNIDATA/BidiMirroring.txt.

If the input
character is a declared as a mirroring character in the
Unicode standard and has a mirrored equivalent. The matching
mirrored character is put in the output, otherwise the input
character itself is put.

RETURNS

If the
character has a mirroring equivalent or not.
