NAME

rewinddir -
reset directory stream

SYNOPSIS

#include

#include

void
rewinddir(DIR *dirp);

DESCRIPTION

The
rewinddir() function resets the position of the
directory stream dirp to the beginning of the
directory.

RETURN VALUE

The
rewinddir() function returns no value.
