NAME

alias &minus;
alias table file of encoding names

SYNOPSIS

/usr/lib/iconv/alias

DESCRIPTION

This file
contains the alias table of encoding names for
iconv_open(3C).

The format of
the alias table is as follows:

"%s %s0, , 

The string
specified for the variant encoding name is case-insensitive.
A line beginning with &rsquo;#&rsquo; is treated as a
comment.
