NAME

mrftopbm -
convert an MRF image to PBM format

SYNOPSIS

mrftopbm
[ -a ] [ input.mrf ]

DESCRIPTION

This program is
part of Netpbm(1).

mrftopbm
converts an MRF image to PBM format.

mrftopbm
takes the MRF image from the file named by the
input.mrf argument, or Standard Input if you
don&rsquo;t specify input.mrf. The output goes to
Standard Output.

For more
information about mrf, see the MRF
specification" (1).

OPTIONS

In addition to
the options common to all programs based on libnetpbm (most
notably -quiet, see
 Common Options ),
mrftopbm recognizes the following command line
option:

-a

causes mrftopbm to include the edges, if any, in
the output PBM. This may help when debugging a
compressor&rsquo;s edge optimization.
