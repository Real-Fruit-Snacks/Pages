NAME

pbmtoatk -
convert a PBM image to a Andrew Toolkit raster object

SYNOPSIS

pbmtoatk
[pbmfile]

DESCRIPTION

This program is
part of Netpbm(1).

pbmtoatk
reads a PBM image as input and produces a Andrew Toolkit
raster object as output.

OPTIONS

There are no
command line options defined specifically for
pbmtoatk, but it recognizes the options common to all
programs based on libnetpbm (See
 Common Options
.)
