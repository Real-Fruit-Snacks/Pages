NAME

gh-pr-close -
Close a pull request

SYNOPSIS

gh pr close
{ |  | }
[flags]

OPTIONS

-c,
--comment 

Leave a closing comment

-d,
--delete-branch

Delete the local and remote
branch after close

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

SEE ALSO

gh-pr(1)
