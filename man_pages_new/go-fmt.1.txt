NAME

go-fmt - gofmt
(reformat) package sources

SYNOPSIS

go fmt
[-n] [-x] [packages]

DESCRIPTION

Fmt runs the
command &rsquo;gofmt -l -w&rsquo; on the packages named by
the import paths. It prints the names of the files that are
modified.

For more about
gofmt, see &rsquo;go doc cmd/gofmt&rsquo;. 
For more about specifying packages, see
go-packages(7).

-n

The -n flag prints commands that would be executed.

-x

The -x flag prints commands as they are executed.

-mod

The -mod flag&rsquo;s value sets which module download
mode to use: readonly or vendor. See &rsquo;go help
modules&rsquo; for more.

To run gofmt
with specific options, run gofmt itself.

SEE ALSO

go-fix(1),
go-vet(1).
