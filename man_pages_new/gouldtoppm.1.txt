NAME

gouldtoppm -
convert Gould scanner file into a PPM image

SYNOPSIS

gouldtoppm
[gouldfile]

DESCRIPTION

This program is
part of Netpbm(1).

gouldtoppm
reads a file produced by the Gould scanner as input and
produces a PPM image as output.

OPTIONS

There are no
command line options defined specifically for
gouldtoppm, but it recognizes the options common to
all programs based on libnetpbm (See
 Common Options
.)
