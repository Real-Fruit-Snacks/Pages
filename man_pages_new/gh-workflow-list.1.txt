NAME

gh-workflow-list
- List workflows

SYNOPSIS

gh workflow
list [flags]

DESCRIPTION

List workflow
files, hiding disabled workflows by default.

OPTIONS

-a,
--all

Include disabled workflows

-q, --jq

Filter JSON output using a jq
expression

--json

Output JSON with the specified
fields

-L, --limit
 (default 50)

Maximum number of workflows to
fetch

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

SEE ALSO

gh-workflow(1)
