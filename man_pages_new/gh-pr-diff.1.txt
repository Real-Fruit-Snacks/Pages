NAME

gh-pr-diff -
View changes in a pull request

SYNOPSIS

gh pr diff
[ |  | ]
[flags]

DESCRIPTION

View changes in
a pull request.

Without an
argument, the pull request that belongs to the current
branch is selected.

With
--web flag, open the pull request diff in a web
browser instead.

OPTIONS

--color
 (default "auto")

Use color in diff output:
{always|never|auto}

--name-only

Display only names of changed
files

--patch

Display diff in patch
format

-w, --web

Open the pull request diff in
the browser

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

SEE ALSO

gh-pr(1)
