NAME

pwdx - report
current working directory of a process

SYNOPSIS

pwdx
[options] pid [...]

OPTIONS

-V,
--version

Output version information and
exit.

-h, --help

Output help screen and
exit.

SEE ALSO

ps(1),
pgrep(1)

STANDARDS

No standards
apply, but pwdx looks an awful lot like a SunOS
command.
