NAME

qrttoppm -
convert output from the QRT ray tracer to a PPM image

SYNOPSIS

qrttoppm

[qrtfile]

DESCRIPTION

This program is
part of Netpbm(1).

qrttoppm
reads a QRT file as input and produces a PPM image as
output.

OPTIONS

There are no
command line options defined specifically for
qrttoppm, but it recognizes the options common to all
programs based on libnetpbm (See
 Common Options
.)
