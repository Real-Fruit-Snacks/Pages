NAME

ppmtoleaf -
convert PPM image to Interleaf image format

SYNOPSIS

ppmtoleaf

[ppmfile]

DESCRIPTION

This program is
part of Netpbm(1).

ppmtoleaf
reads an Interleaf image file as input and generates a PPM
image as output.

Interleaf is a
now-defunct (actually purchased ca. 2000 by BroadVision)
technical publishing software company.

OPTIONS

There are no
command line options defined specifically for
ppmtoleaf, but it recognizes the options common to
all programs based on libnetpbm (See
 Common Options
.)
