NAME

gh-codespace-edit
- Edit a codespace

SYNOPSIS

gh codespace
edit [flags]

OPTIONS

-c,
--codespace 

Name of the codespace

-d, --display-name

Set the display name

-m, --machine

Set hardware specifications for
the VM

-R, --repo

Filter codespace selection by
repository name (user/repo)

--repo-owner

Filter codespace selection by
repository owner (username or org)

SEE ALSO

gh-codespace(1)
