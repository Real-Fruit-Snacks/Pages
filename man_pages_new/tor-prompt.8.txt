NAME

tor-prompt - a
stem direct access implementation to Tor&rsquo;s control
interface

SYNOPSIS

tor-prompt
[options]

DESCRIPTION

tor-prompt
acts like a standard python interpreter with a Tor
connection available via your &rsquo;controller&rsquo;
variable

List of
available options:

/help

provides information for interpreter and tor
commands 

/events

prints events that we&rsquo;ve
received

/info

general information for a relay

/python

enable or disable support for
running python commands

/quit

shuts down the interpreter

For more information use

&rsquo;/help
[OPTION]&rsquo;
