NAME

rmdir - remove
empty directories

SYNOPSIS

rmdir
[OPTION]... DIRECTORY...

DESCRIPTION

Remove the
DIRECTORY(ies), if they are empty. 
--ignore-fail-on-non-empty

ignore each failure to remove a
non-empty directory

-p, --parents

remove DIRECTORY and its
ancestors; e.g., &rsquo;rmdir -p a/b&rsquo; is
similar to &rsquo;rmdir a/b a&rsquo;

-v, --verbose

output a diagnostic for every
directory processed

--help

display this help and exit

--version

output version information and
exit
