NAME

neotoppm -
convert an Atari Neochrome .neo into a PPM image

SYNOPSIS

neotoppm
[neofile]

DESCRIPTION

This program is
part of Netpbm(1).

neotoppm
reads an Atari Neochrome .neo file as input and produces a
PPM image as output.

OPTIONS

There are no
command line options defined specifically for
neotoppm, but it recognizes the options common to all
programs based on libnetpbm (See
 Common Options
.)

ppmtoneo(1),
