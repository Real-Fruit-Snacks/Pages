NAME

setlogcons -
Send kernel messages to console N

SYNOPSIS

setlogcons
N

DESCRIPTION

The command
setlogcons N sets all kernel messages to the
console specified as N.

By default
kernel messages are sent to the current console.

To change the
level of messages sent, use dmesg
