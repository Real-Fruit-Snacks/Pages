NAME

iso_8859-1 -
ISO 8859-1 character set encoded in octal, decimal, and
hexadecimal

DESCRIPTION

The ISO 8859
standard includes several 8-bit extensions to the ASCII
character set (also known as ISO 646-IRV). ISO 8859-1
encodes the characters used in many West European
languages.
