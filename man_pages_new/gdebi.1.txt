NAME

gdebi &minus;
Simple tool to install deb files

SYNOPSIS

gdebi
[package.deb]...

DESCRIPTION

gdebi lets you
install local deb packages resolving and installing its
dependencies. apt does the same, but only for remote (http,
ftp) located packages. It can also resolve build-depends of
debian/control files.

OPTIONS

&minus;&minus;version

Show program&rsquo;s version
number and exit.

&minus;h,
&minus;&minus;help

Show this help message and
exit.

&minus;&minus;n,
&minus;&minus;non&minus;interactive

Run non-interactive
(dangerous!).

&minus;&minus;o
APT_OPTS,
&minus;&minus;option=APT_OPTS

Set an APT configuration
option.

&minus;&minus;q,
&minus;&minus;quiet

Do not show progress
information.

&minus;&minus;apt&minus;line

Simulate only and print a
apt-get install compatible line to stderr.

&minus;&minus;root=ROOTDIR

Use alternative root dir.

EXAMPLES

gdebi
foo_1.0_all.deb 
gdebi foo-1.0/debian/control
