NAME

podman-healthcheck-run
- Run a container healthcheck

SYNOPSIS

podman
healthcheck run container

DESCRIPTION

Runs the
healthcheck command defined in a running container manually.
The resulting error codes are defined as follows:

&bull;

0 = healthcheck command succeeded

&bull;

1 = healthcheck command failed

&bull;

125 = an error has occurred

Possible errors
that can occur during the healthcheck are: * unable to find
the container * container has no defined healthcheck *
container is not running

OPTIONS

--help

Print usage statement

EXAMPLES

$ podman
healthcheck run mywebapp

SEE ALSO

podman(1),
podman-healthcheck(1)
