NAME

dev&minus;ram&minus;control
&minus; checks features of DVD-RAM discs

SYNOPSIS

dvd&minus;ram&minus;control
[&minus;arre] [&minus;awre] [&minus;rdonly|&minus;rdwr]
/dev/dvd

DESCRIPTION

The
dvd&minus;ram&minus;control command allows the user
to check/set features of DVD-RAM discs. One such information
is the Persistent Write Protection, controlling if the disc
can be in read-only mode or if both reads and writes are
enabled. It also allows the user to check and or set if
Automatic Write Relocation Enable (AWRE) as well as
Automatic Read Relocation Enable (ARRE) are enabled or not
for a given media.
