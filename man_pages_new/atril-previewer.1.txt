NAME

atril-previewer
- show print preview for a document

SYNOPSIS

atril-previewer
[file]

DESCRIPTION

atril-previewer
is a MATE program to show print preview dialog for files
supported by atril.

OPTIONS

atril-previewer
obeys all normal GTK+ command line options.

SEE ALSO

atril(1),
gtk-options(7).

https://mate-desktop.org/
