NAME

gh-project-create
- Create a project

SYNOPSIS

gh project
create [flags]

OPTIONS

--format

Output format: {json}

-q, --jq

Filter JSON output using a jq
expression

--owner

Login of the owner. Use
"@me" for the current user.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

--title

Title for the project

EXAMPLE

# create a new
project owned by login monalisa 
gh project create --owner monalisa --title "a new
project"

SEE ALSO

gh-project(1)
