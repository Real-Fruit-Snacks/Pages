NAME

mate-power-statistics
- MATE power statistics GUI

SYNOPSIS

mate-power-statistics
[ --verbose ] [ --help ]

DESCRIPTION

mate-power-statistics
is the gui program for the mate power management
infrastructure.

It allows users
to visualize the power consumption of laptop hardware.

OPTIONS

--help

Show summary of options.

--verbose

Show extra debugging.

SEE ALSO

mate-power-manager
(1), mate-power-preferences (1).
