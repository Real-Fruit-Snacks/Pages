NAME

wwsympa,
wwsympa.fcgi - WWSympa, Sympa’s web interface

DESCRIPTION

This FastCGI
script completely handles all aspects of the Sympa web
interface.

To know details
on WWSympa, see Sympa Administration Manual:
https://www.sympa.community/manual/>.
