NAME

gh-extension-create
- Create a new extension

SYNOPSIS

gh extension
create [] [flags]

OPTIONS

--precompiled

Create a precompiled extension.
Possible values: go, other

EXAMPLE

# Use
interactively 
gh extension create

# Create a
script-based extension 
gh extension create foobar

# Create a Go
extension 
gh extension create --precompiled=go foobar

# Create a
non-Go precompiled extension 
gh extension create --precompiled=other foobar

SEE ALSO

gh-extension(1)
