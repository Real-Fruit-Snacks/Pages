NAME
       Dpkg::OpenPGP::KeyHandle - OpenPGP key handle support

DESCRIPTION
       This module provides a class to hold an OpenPGP key handle, as a way
       for the code to refer to a key in an independent way.

       Note: This is a private module, its API can change at any time.
