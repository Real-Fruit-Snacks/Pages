NAME
       Squirrel::Role - Use Mouse::Role, unless Moose::Role is already loaded.
       (DEPRECATED)

SYNOPSIS
           use Squirrel::Role;

DEPRECATION
       "Squirrel::Role" is deprecated. "Any::Moose" provides the same
       functionality, but better. :)
