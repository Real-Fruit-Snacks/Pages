NAME
       Image::ExifTool::GIMP - Read meta information from GIMP XCF images

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to read
       meta information from GIMP (GNU Image Manipulation Program) XCF
       (eXperimental Computing Facility) images.  This is the native image
       format used by the GIMP software.
