NAME

gh-project-field-list
- List the fields in a project

SYNOPSIS

gh project
field-list number [flags]

OPTIONS

--format

Output format: {json}

-q, --jq

Filter JSON output using a jq
expression

-L, --limit
 (default 30)

Maximum number of fields to
fetch

--owner

Login of the owner. Use
"@me" for the current user.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

EXAMPLE

# list fields
in the current userâ€™s project "1" 
gh project field-list 1 --owner "@me"
