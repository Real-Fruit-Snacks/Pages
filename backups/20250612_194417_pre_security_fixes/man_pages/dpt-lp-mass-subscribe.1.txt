NAME

lp-mass-subscribe
- subscribe to many packages on Launchpad

SYNOPSIS

 $ get-ubuntu-packages | lp-mass-subscribe

DESCRIPTION

This script
will connect to Ubuntu launchpad and subscribe
pkg-perl-maintainers to the list of packages which it
expects on the standard input, one package per line. It is
usually used together with
get-ubuntu-packages(1).
