NAME

marco-window-demo
- demo of window features

SYNOPSIS

marco-window-demo

DESCRIPTION

This program
demonstrates various kinds of windows that window managers
and window manager themes should handle.

Be sure to tear
off the menu and toolbar, those are also a special kind of
window.
