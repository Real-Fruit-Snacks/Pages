NAME

lat_sig -
select benchmark

SYNOPSIS

lat_ctx
[ -P  ] [ -W
 ] [ -N  ]
install|catch|prot [ file ]

DESCRIPTION

lat_sig
measures the time to install and catch signals. It can also
measure the time to catch a protection fault.
