NAME

curl_url_dup -
duplicate a CURLU handle

SYNOPSIS

#include

CURLU
*curl_url_dup(CURLU *inhandle);

DESCRIPTION

Duplicates a
given CURLU inhandle and all its contents and returns
a pointer to a new CURLU handle. The new handle also needs
to be freed with curl_url_cleanup(3).

RETURN VALUE

Returns a new
handle or NULL if out of memory.

EXAMPLE

CURLUcode rc;

CURLU *url = curl_url(); 
CURLU *url2; 
rc = curl_url_set(url, CURLUPART_URL,
"https://example.com", 0); 
if(!rc) { 
url2 = curl_url_dup(url); /* clone it! */ 
curl_url_cleanup(url2); 
} 
curl_url_cleanup(url);

AVAILABILITY

Added in curl
7.62.0
