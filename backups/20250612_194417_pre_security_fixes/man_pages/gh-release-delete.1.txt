NAME

gh-release-delete
- Delete a release

SYNOPSIS

gh release
delete  [flags]

OPTIONS

--cleanup-tag

Delete the specified tag in
addition to its release

-y, --yes

Skip the confirmation
prompt

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

SEE ALSO

gh-release(1)
