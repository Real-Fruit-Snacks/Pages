NAME

cerfcx, erfcx
− underflow−compensated complementary error
function

SYNOPSIS

#include
>

double
_Complex cerfcx ( double _Complex z );

double erfcx
( double x );

DESCRIPTION

The function
cerfcx is an underflow-compensated variant of the
complex error function: erfcx(z) = exp(z^2) erfc(z).

The function
erfcx takes a real argument and returns a real
result.

RESOURCES

Project web
site: http://apps.jcns.fz−juelich.de/libcerf

REFERENCES

The
implementation of cerfcx is a thin wrapper around
Faddeeva’s function w_of_z.

The
implementation of ercx is self-contained, and
improves upon the SLATEC DERFC function (or
an erfcx function derived therefrom) or Cody’s
CALERF function (from netlib.org/specfun),
while retaining near machine precision in accuracy.
