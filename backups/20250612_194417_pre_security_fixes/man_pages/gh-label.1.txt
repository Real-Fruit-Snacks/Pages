NAME

gh-label -
Manage labels

SYNOPSIS

gh label
 [flags]

DESCRIPTION

Work with
GitHub labels.

AVAILABLE COMMANDS

gh-label-clone(1)

Clones labels from one
repository to another

gh-label-create(1)

Create a new label

gh-label-delete(1)

Delete a label from a
repository

gh-label-edit(1)

Edit a label

gh-label-list(1)

List labels in a repository

OPTIONS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

SEE ALSO

gh(1)
