NAME

proxy - Display
the proxy server that should be used to reach a given a
network resource

SYNOPSIS

proxy

DESCRIPTION

Display the
proxy server that should be used to reach a given a network
resource.

libproxy is a
library that provides automatic proxy configuration
management using different backends.

EXAMPLE

echo
http://www.example.com/ | proxy 
http://webcache:3128 direct://
