NAME

fribidi_get_bracket
- get bracketed character

SYNOPSIS

#include

FriBidiBracketType
fribidi_get_bracket(FriBidiChar ch);

PARAMETERS

FriBidiChar
ch

Input character.

DESCRIPTION

This function
finds the bracketed equivalent of a character as defined in
the file BidiBrackets.txt of the Unicode Character Database
available at
http://www.unicode.org/Public/UNIDATA/BidiBrackets.txt.

If the input
character is a declared as a brackets character in the
Unicode standard and has a bracketed equivalent. The
matching bracketed character is put in the output, otherwise
the input character itself is put.

RETURNS

The bracket
type of the character. Use the
FRIBIDI_IS_BRACKET(FriBidiBracketType) to test if it is a
valid property.
