NAME

marco-message -
A command to send messages to Marco.

SYNOPSIS

marco-message
[OPTIONS]

DESCRIPTION

The
marco-message command is used to perform some simple
window manager tasks while Marco is running.

OPTIONS

restart

Restart a currently running
instance of marco.

reload-theme

Reload the current window
border theme specified in the dconf database.
(/org/mate/marco/general/theme)

enable-keybindings

Enable all keybindings
specified in the dconf database. (/org/mate/marco/)

disable-keybindings

Disable all keybindings
specified in the dconf database. (/org/mate/marco/)

toggle-verbose

Enable or Disable debugging
messages for marco.
