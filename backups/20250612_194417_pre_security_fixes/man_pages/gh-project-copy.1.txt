NAME

gh-project-copy
- Copy a project

SYNOPSIS

gh project
copy [] [flags]

OPTIONS

--drafts

Include draft issues when
copying

--format

Output format: {json}

-q, --jq

Filter JSON output using a jq
expression

--source-owner

Login of the source owner. Use
"@me" for the current user.

--target-owner

Login of the target owner. Use
"@me" for the current user.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

--title

Title for the new project

EXAMPLE

# copy project
"1" owned by monalisa to github 
gh project copy 1 --source-owner monalisa --target-owner
github --title "a new project"

SEE ALSO
