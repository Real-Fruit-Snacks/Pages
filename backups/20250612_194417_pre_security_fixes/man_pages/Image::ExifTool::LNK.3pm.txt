NAME
       Image::ExifTool::LNK - Read MS Shell Link (.LNK) meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information MS Shell Link (Windows shortcut) files.
