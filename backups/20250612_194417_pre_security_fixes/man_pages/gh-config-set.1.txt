NAME

gh-config-set -
Update configuration with a value for the given key

SYNOPSIS

gh config
set   [flags]

OPTIONS

-h,
--host 

Set per-host setting

EXAMPLE

$ gh config set
editor vim 
$ gh config set editor "code --wait" 
$ gh config set git_protocol ssh --host github.com 
$ gh config set prompt disabled

gh-config(1)
