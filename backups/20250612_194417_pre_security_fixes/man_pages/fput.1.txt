NAME

fput −
upload files to the FSP server

SYNOPSIS

fput [
-p ] filename ...

DESCRIPTION

When used
without arguments, a list of file names is read from the
standard input. When arguments are given, the arguments are
taken to be the list of file names. For each file named in
the list, fput copies the file from the
client’s current working directory in the client
machine to the FSP database.

ARGUMENTS

-p --
preserve date/time of original file.

ENVIRONMENT

See fsp_env(7)
for list of used environment variables.
