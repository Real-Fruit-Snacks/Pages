NAME

ppmtopi1 -
convert a PPM image into an Atari Degas .pi1 file

SYNOPSIS

ppmtopi1

[ppmfile]

DESCRIPTION

This program is
part of Netpbm(1).

ppmtopi1
reads a PPM image as input and produces an Atari Degas .pi1
file as output.

OPTIONS

There are no
command line options defined specifically for
ppmtopi1, but it recognizes the options common to all
programs based on libnetpbm (See
 Common Options
.)
