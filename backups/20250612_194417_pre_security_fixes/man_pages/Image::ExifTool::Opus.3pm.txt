NAME
       Image::ExifTool::Opus - Read Ogg Opus audio meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information from Ogg Opus audio files.
