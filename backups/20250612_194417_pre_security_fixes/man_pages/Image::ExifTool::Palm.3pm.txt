NAME
       Image::ExifTool::Palm - Read Palm Database files

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains code to extract metadata from Palm database files
       (PDB and PRC extensions), Mobipocket electronic books (MOBI), and
       Amazon Kindle KF7 and KF8 books (AZW and AZW3).
