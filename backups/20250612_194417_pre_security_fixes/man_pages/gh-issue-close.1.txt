NAME

gh-issue-close
- Close issue

SYNOPSIS

gh issue
close { | } [flags]

OPTIONS

-c,
--comment 

Leave a closing comment

-r, --reason

Reason for closing:
{completed|not planned}

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

SEE ALSO

gh-issue(1)
