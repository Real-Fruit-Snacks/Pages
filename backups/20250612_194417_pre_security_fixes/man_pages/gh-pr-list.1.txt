NAME

gh-pr-list -
List pull requests in a repository

SYNOPSIS

gh pr list
[flags]

DESCRIPTION

List pull
requests in a GitHub repository.

The search
query syntax is documented here:

⟨
https://docs.github.com/en/search-github/searching-on-github/searching-issues-and-pull-requests⟩

OPTIONS

--app

Filter by GitHub App author

-a, --assignee

Filter by assignee

-A, --author

Filter by author

-B, --base

Filter by base branch

-d, --draft

Filter by draft state

-H, --head

Filter by head branch

-q, --jq

Filter JSON output using a jq
expression

--json

Output JSON with the specified
fields

-l, --label

Filter by label

-L, --limit
 (default 30)

Maximum number of items to
fetch

-S, --search

Search pull requests with
query

-s, --state
 (default "open")

Filter by state:
{open|closed|merged|all}

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

-w, --web

List pull requests in the web
browser

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

EXAMPLE

List PRs
authored by you 
$ gh pr list --author "@me"

List only PRs
with all of the given labels 
$ gh pr list --label bug --label "priority 1"

Filter PRs
using search syntax 
$ gh pr list --search "status:success
review:required"

Find a PR that
introduced a given commit 
$ gh pr list --search "" --state
merged
