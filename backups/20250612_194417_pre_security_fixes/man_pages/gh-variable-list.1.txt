NAME

gh-variable-list
- List variables

SYNOPSIS

gh variable
list [flags]

DESCRIPTION

List variables
on one of the following levels: - repository (default):
available to GitHub Actions runs or Dependabot in a
repository - environment: available to GitHub Actions runs
for a deployment environment in a repository - organization:
available to GitHub Actions runs or Dependabot within an
organization

OPTIONS

-e,
--env 

List variables for an
environment

-q, --jq

Filter JSON output using a jq
expression

--json

Output JSON with the specified
fields

-o, --org

List variables for an
organization

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format
