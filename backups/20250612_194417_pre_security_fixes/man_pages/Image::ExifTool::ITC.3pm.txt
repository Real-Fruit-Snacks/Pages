NAME
       Image::ExifTool::ITC - Read iTunes Cover Flow meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains the routines required by Image::ExifTool to read
       meta information (including artwork images) from iTunes Cover Flow
       files.
