NAME

cerf, cerfc
− complex error functions

SYNOPSIS

#include
>

double
_Complex cerf ( double _Complex z );

double
_Complex cerfc ( double _Complex z );

DESCRIPTION

The function
cerf is the complex version of the error function:
erf(z) = 2/sqrt(pi) * integral from 0 to z of
exp(−t*t) dt.

The
complementary complex error function cerfc is defined
as erfc(z) = 1−cerf(z).

RESOURCES

Project web
site: http://apps.jcns.fz−juelich.de/libcerf

REFERENCES

The
computations are based on Faddeeva’s function
w_of_z.
