NAME

gh-issue-list -
List issues in a repository

SYNOPSIS

gh issue
list [flags]

DESCRIPTION

List issues in
a GitHub repository.

The search
query syntax is documented here:

⟨
https://docs.github.com/en/search-github/searching-on-github/searching-issues-and-pull-requests⟩

OPTIONS

--app

Filter by GitHub App author

-a, --assignee

Filter by assignee

-A, --author

Filter by author

-q, --jq

Filter JSON output using a jq
expression

--json

Output JSON with the specified
fields

-l, --label

Filter by label

-L, --limit
 (default 30)

Maximum number of issues to
fetch

--mention

Filter by mention

-m, --milestone

Filter by milestone number or
title

-S, --search

Search issues with query

-s, --state
 (default "open")

Filter by state:
{open|closed|all}

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

-w, --web

List issues in the web
browser

OPTIONS INHERITED FROM PARENT COMMANDS

-R,
--repo 

Select another repository using
the [HOST/]OWNER/REPO format

EXAMPLE

$ gh issue list
--label "bug" --label "help wanted" 
$ gh issue list --author monalisa 
$ gh issue list --assignee "@me" 
$ gh issue list --milestone "The big 1.0" 
$ gh issue list --search "error no:assignee
sort:created-asc"
