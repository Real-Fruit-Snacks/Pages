NAME

gh-project-field-delete
- Delete a field in a project

SYNOPSIS

gh project
field-delete [flags]

OPTIONS

--format

Output format: {json}

--id 

ID of the field to delete

-q, --jq

Filter JSON output using a jq
expression

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

SEE ALSO

gh-project(1)
