AUTH_GETVAL(3)

NAME

auth_getval —
functions for reading values from /etc/auth.conf

LIBRARY

System Utilities Library
(libutil, −lutil)

SYNOPSIS

#include 

#include 

char
*

auth_getval(const char *name);

DESCRIPTION

The function
auth_getval() returns the value associated with the
field called name or NULL if no such field is found
or the auth file cannot be opened.

SEE ALSO

properties_free(3),
properties_read(3), property_find(3), auth.conf(5)

FILES

/etc/auth.conf contains
the name=value pairs looked up by auth_getval().
