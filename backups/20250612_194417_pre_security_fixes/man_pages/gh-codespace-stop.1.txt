NAME

gh-codespace-stop
- Stop a running codespace

SYNOPSIS

gh codespace
stop [flags]

OPTIONS

-c,
--codespace 

Name of the codespace

-o, --org

The login handle of the
organization (admin-only)

-R, --repo

Filter codespace selection by
repository name (user/repo)

--repo-owner

Filter codespace selection by
repository owner (username or org)

-u, --user

The username to stop codespace
for (used with --org)

SEE ALSO

gh-codespace(1)
