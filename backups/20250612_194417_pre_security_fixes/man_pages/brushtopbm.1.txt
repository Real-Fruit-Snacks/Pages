NAME

brushtopbm -
convert a doodle brush file into a PBM image

SYNOPSIS

brushtopbm
[brushfile]

DESCRIPTION

This program is
part of Netpbm(1).

brushtopbm
reads a Xerox doodle brush file as input. and produces a
portable bitmap as output.

Note that there
is currently no pbmtobrush tool.

OPTIONS

There are no
command line options defined specifically for
brushtopbm, but it recognizes the options common to
all programs based on libnetpbm (See
 Common Options
.)
