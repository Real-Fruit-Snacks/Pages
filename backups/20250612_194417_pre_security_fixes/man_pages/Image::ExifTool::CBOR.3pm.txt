NAME
       Image::ExifTool::CBOR - Read CBOR format metadata

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool read
       Concise Binary Object Representation (CBOR) formatted metadata, used by
       the C2PA specification.
