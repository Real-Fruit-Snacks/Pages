NAME
       Image::ExifTool::DjVu - Read DjVu meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information from DjVu images.  Parsing of the DjVu IFF structure
       is done by Image::ExifTool::AIFF.
