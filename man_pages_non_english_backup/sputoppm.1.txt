NAME
       sputoppm - convert an Atari uncompressed Spectrum file to a PPM image

SYNOPSIS
       sputoppm

       [spufile]

DESCRIPTION
       This program is part of Netpbm(1).

       sputoppm  reads  an  Atari uncompressed Spectrum file as input and pro‐
       duces a PPM image as output.

OPTIONS
       There are no command line options defined  specifically  for  sputoppm,
       but it recognizes the options common to all programs based on libnetpbm
       (See
        Common Options ⟨index.html#commonoptions⟩ .)
