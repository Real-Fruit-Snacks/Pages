NAME
       Image::ExifTool::DNG.pm - Read DNG-specific information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains routines required by Image::ExifTool to process
       information in DNG (Digital Negative) images.
