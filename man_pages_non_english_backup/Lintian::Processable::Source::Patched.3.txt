NAME
       Lintian::Processable::Source::Patched - access to sources with Debian
       patches applied

SYNOPSIS
           use Lintian::Processable;

DESCRIPTION
       Lintian::Processable::Source::Patched provides an interface to
       collected data about patched sources.

INSTANCE METHODS
       patched
           Returns a index object representing a patched source tree.

AUTHOR

       Lintian.
