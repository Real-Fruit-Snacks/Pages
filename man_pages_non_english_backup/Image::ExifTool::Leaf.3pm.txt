NAME
       Image::ExifTool::Leaf - Read Creo Leaf EXIF meta information

SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.

DESCRIPTION
       This module contains definitions required by Image::ExifTool to
       interpret meta information from Leaf digital camera backs written by
       Creo Leaf Capture.
