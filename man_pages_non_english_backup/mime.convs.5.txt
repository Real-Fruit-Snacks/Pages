NOME
       mime.convs  - ficheiro de conversão do tipo mime para cups (descontinu‐
       ado)

DESCRIÇÃO
       O ficheiro mime.convs define os filtros que estão disponíveis para con‐
       verter ficheiros de um formato para outro. Os filtros standard suportam
       texto, PDF, PostScript, e muitos outros tipos de ficheiros de imagem.

       Filtros adicionais são especificados em ficheiros com a extensão .convs
       no directório de configuração do CUPS.

       Cada  linha  no  ficheiro  mime.convs  é um comentário, linha vazia, ou
       linha de filtro. AS linhas de comentários começam com  o  caractere  #.
       Linhas  de  filtros especificam os tipos MIME de fonte e destino junta‐
       mente com um custo relativo associado ao filtro e o filtro a correr:

           source/type destination/type cost filter

       O campo source/type especifica o tipo de media MIME fonte que é consum‐
       ido pelo filtro.

       O campo destination/type especifica o tipo de media MIME de destino que
       é produzido pelo filtro.

       O campo cost especifica o custo relativo de correr o filtro.  Um  valor
       de  100  significa que o filtro usa grandes quantidades de recursos en‐
       quanto um valor de 0 significa que o filtro usa muito poucos recursos.

       O campo filter especifica o nome de ficheiro  do  programa  filtro.  Os
       nomes de ficheiros são relativos ao directório de filtros do CUPS.

FICHEIROS
       /etc/cups - Directório típico de configuração do CUPS.
       /usr/lib/cups/filter -  Directório típico de filtros do CUPS.
       /usr/libexec/cups/filter - Directório de filtros do CUPS no macOS.

EXEMPLOS
       Definir  um filtro que converta documentos PostScript no formato Raster
       do CUPS:

           application/vnd.cups-postscript application/vnd.cups-raster 50 pstoraster

NOTAS
       Os filtros do CUPS estão descontinuados e não serão mais suportados num
       lançamento  futuro  do CUPS.  As impressoras que não suportem IPP podem
       ser suportadas usando aplicações como o ippeveprinter(1).
