NAME
       Image::ExifTool::Red - Read Redcode R3D video files

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains routines required by Image::ExifTool to read
       metadata from Redcode R3D version 1 and 2 video files.
