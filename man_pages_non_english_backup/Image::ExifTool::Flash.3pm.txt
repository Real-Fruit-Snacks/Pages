NAME
       Image::ExifTool::Flash - Read Shockwave Flash meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to read
       SWF (Shockwave Flash) and FLV (Flash Video) files.

NOTES
       Flash Video AMF3 support has not yet been added because I haven't yet
       found a FLV file containing AMF3 information.  If someone sends me a
       sample then I will add AMF3 support.
