NAME
       Image::ExifTool::PNG - Read and write PNG meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains routines required by Image::ExifTool to read and
       write PNG (Portable Network Graphics), MNG (Multi-image Network
       Graphics) and JNG (JPEG Network Graphics) images.
