NAME

gh-project-close
- Close a project

SYNOPSIS

gh project
close [] [flags]

OPTIONS

--format

Output format: {json}

-q, --jq

Filter JSON output using a jq
expression

--owner

Login of the owner. Use
"@me" for the current user.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

--undo

Reopen a closed project

EXAMPLE

# close project
"1" owned by monalisa 
gh project close 1 --owner monalisa

# reopen closed
project "1" owned by github 
gh project close 1 --owner github --undo
