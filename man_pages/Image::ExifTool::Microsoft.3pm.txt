NAME
       Image::ExifTool::Microsoft - Definitions for custom Microsoft tags

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to
       interpret Microsoft-specific EXIF and XMP tags, and routines to
       read/write Microsoft Xtra tags in videos.
