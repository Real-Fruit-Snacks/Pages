NOME
       ppdi - importa ficheiros ppd (descontinuado)

RESUMO
       ppdi  [  -I  include-directory  ]  [  -o  source-file  ] ppd-file [ ...
       ppd-file ]

DESCRIÇÃO
       ppdi importa um ou mais ficheiros PPD para um ficheiro fonte de  compi‐
       lador  de  PPD. As várias linguagens do mesmo ficheiro PPD são fundidas
       numa única definição de impressora para facilitar  alterações  precisas
       para todas as localizações. Este programa está descontinuado e será re‐
       movido num lançamento futuro do CUPS.

OPÇÕES
       ppdi suporta as seguintes opções:

       -I include-directory
            Especifica  um  directório  de  inclusão  alternativo.  Podem  ser
            fornecidas  múltiplas  opções  -I  para adicionar directórios adi‐
            cionais.

       -o source-file
            Especifica o ficheiro fonte PPD a actualizar. Se o ficheiro  fonte
            não  existir,  é  criado  um novo ficheiro fonte. Caso contrário o
            ficheiro existente é fundido com os novos ficheiro(s) PPD na linha
            de  comandos. Se não for especificado nenhum ficheiro fonte, é us‐
            ado o nome de ficheiro ppdi.drv.

NOTAS
       Ficheiros PPD estão descontinuados e não irão ser mais  suportados  num
       futuro  lançamento  do CUPS. Impressoras que não suportem IPP podem ser
       suportadas usando aplicações como a ippeveprinter(1).
