NAME
       Lintian::Data::Stylesheet - Lintian interface to CSS style sheets

SYNOPSIS
           use Lintian::Data::Stylesheet;

DESCRIPTION
       This module provides a way to load data files to CSS style sheets

       location
       "css"
       load
       refresh
