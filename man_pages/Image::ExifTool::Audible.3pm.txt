NAME
       Image::ExifTool::Audible - Read meta information from Audible audio
       books

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to read
       meta information from Audible audio books.
