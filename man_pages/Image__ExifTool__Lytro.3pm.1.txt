NAME
       Image::ExifTool::Lytro - Read Lytro LFP files

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains routines required by Image::ExifTool to read
       metadata from Lytro Light Field Picture (LFP) files.
