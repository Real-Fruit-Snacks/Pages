NAME

cerfi, erfi
&minus; imaginary error function

SYNOPSIS

#include
>

double
_Complex cerfi ( double _Complex z );

double erfi
( double x );

DESCRIPTION

The function
cerfi returns an error function rotated in the
complex plane, erfi(z) = &minus;i erf(iz).

The function
erfi takes a real argument and returns a real
result.

RESOURCES

Project web
site: http://apps.jcns.fz&minus;juelich.de/libcerf

REFERENCES

The
implementation of cerfi and erfi is trivially
based on the functions cerf and im_w_of_x.
