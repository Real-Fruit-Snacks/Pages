NOME
       /etc/deluser.conf  -  ficheiro  de  configuração para deluser(8) e del‐
       group(8).

DESCRIÇÃO
       O ficheiro /etc/deluser.conf contém  predefinições  para  os  programas
       deluser(8)  e delgroup(8). Cada linha contém um único par de valores no
       formato opção = valor. São permitidas cotações simples ou duplas a  de‐
       limitar  o  valor, assim como o espaço em branco a rodear o sinal igual
       (=). As linhas comentários têm de ter um cardinal (#) na primeira  col‐
       una.

       deluser(8)  e  delgroup(8)  também  leem  /etc/adduser.conf,  veja  ad‐
       duser.conf(5);  as  definições  em  deluser.conf  podem   sobrepor   as
       definições feitas em adduser.conf.

       As opções de configuração válidas são:

       BACKUP Se  REMOVE_HOME  ou REMOVE_ALL_FILES estiverem activos, todos os
              ficheiros  são  salvaguardados  antes  de  serem  removidos.   O
              ficheiro   de   backup   que   é  criado  usa  por  predefinição
              nome-de-utilizador.tar(.gz|.bz2) no directório especificado pela
              opção  BACKUP_TO. O método de compressão é escolhido pelo melhor
              que está disponível. Valores podem ser 0 ou 1. A predefinição  é
              0.

       BACKUP_SUFFIX
              Seleciona o algoritmo de compressão para um backup de directório
              home. Pode ser definido para qualquer sufixo reconhecido por tar
              --auto-compress.  A predefinição é .gz.

       BACKUP_TO
              Se BACKUP estiver activo, BACKUP_TO especifica o directório onde
              a salvaguarda é escrita. A predefinição é o directório actual.

       EXCLUDE_FSTYPES
              Uma expressão regular que descreve todos os tipos de sistemas de
              ficheiros que devem ser excluídos quando se procura ficheiros de
              um  utilizador   para   serem   apagados.   A   predefinição   é
              "(proc|sysfs|usbfs|devtmpfs|devpts|afs)".

       NO_DEL_PATHS
              Uma  lista  de expressões regulares separadas por espaços. Todos
              os ficheiros a serem apagados no curso de  apagar  o  directório
              home  ou  ficheiros cujo utilizador é dono em qualquer sítio são
              verificados contra cada uma destas expressões regulares. Se  for
              detetada  uma  correspondência, o ficheiro não é apagado. A pre‐
              definição é uma lista de directórios de sistema, deixando apenas
              a  /home.  Assim apenas ficheiros sob /home que pertencem a esse
              utilizador específico irão ser apagados.

       ONLY_IF_EMPTY
              Apenas apaga um grupo se não existirem utilizadores pertencentes
              a este grupo. A predefinição é 0.

       REMOVE_ALL_FILES
              Remove todos os ficheiros no sistema cujo dono seja o utilizador
              a ser removido. Se esta opção estiver  activa,  REMOVE_HOME  não
              tem efeito. O valor pode ser 0 ou 1. A predefinição é 0.

       REMOVE_HOME
              Remove  o  directório home e spool de mail do utilizador que vai
              ser removido. O valor pode ser 0 (não apagar) ou 1  (apagar).  A
              predefinição é 0.

FICHEIROS
       /etc/deluser.conf

VEJA TAMBÉM
       adduser.conf(5), delgroup(8), deluser(8)
