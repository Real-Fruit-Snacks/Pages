NAME

fver &minus;
displays the version number string of the FSP database

SYNOPSIS

fver
[-l]

DESCRIPTION

Without
arguments, fver displays the version string of the
remote FSP server. Otherwise, the version string of the
client utilities and compile time options are displayed.

ENVIRONMENT

See fsp_env(7)
for list of used environment variables.
