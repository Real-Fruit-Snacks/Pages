NAME

fstatcmd, fstat
displays information on remote files and directories

SYNOPSIS

fstatcmd
filename ...

fstat
filename ...

DESCRIPTION

fstatcmd
will display the type (file or directory), name, size in
bytes and creation time of the given remote file. The time
and date of creation will be displayed in ISO format.

Since
user’s command shells generally cannot do the proper
globbing for this program, it is recommended that user
defines a shell alias or function to turn off command shell
globbing before running this program.

csh
example:

alias fstat
\(set noglob\; exec fstatcmd \!\*\)

bash
example:

_fstat() {
fstatcmd "$@" ;set +f;} 
alias fstat=’set -f;_fstat’

for list of used environment variables.
