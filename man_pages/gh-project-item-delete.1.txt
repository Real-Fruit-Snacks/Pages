NAME

gh-project-item-delete
- Delete an item from a project by ID

SYNOPSIS

gh project
item-delete [] [flags]

OPTIONS

--format

Output format: {json}

--id 

ID of the item to delete

-q, --jq

Filter JSON output using a jq
expression

--owner

Login of the owner. Use
"@me" for the current user.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

EXAMPLE

# delete an
item in the current userâ€™s project "1" 
gh project item-delete 1 --owner "@me" --id
