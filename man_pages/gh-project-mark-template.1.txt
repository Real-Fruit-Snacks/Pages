NAME

gh-project-mark-template
- Mark a project as a template

SYNOPSIS

gh project
mark-template [] [flags]

OPTIONS

--format

Output format: {json}

-q, --jq

Filter JSON output using a jq
expression

--owner

Login of the org owner.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

--undo

Unmark the project as a template.

EXAMPLE

# mark the
github org&rsquo;s project "1" as a template 
gh project mark-template 1 --owner "github"

# unmark the
github org&rsquo;s project "1" as a template 
gh project mark-template 1 --owner "github"
--undo
