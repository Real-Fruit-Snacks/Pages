NAME
       Image::ExifTool::JPEG - Definitions for uncommon JPEG segments

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool for some
       uncommon JPEG segments.  For speed reasons, definitions for more common
       JPEG segments are included in the Image::ExifTool module itself.
