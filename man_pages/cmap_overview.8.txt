NAME

cmap_overview
&minus; Overview of the Configuration Map

OVERVIEW

The CMAP
library is used to interact with the configuration database
used by corosync.

The library
provides a mechanism to:

* Create of new
keys

* Change
existing keys

* Remove
keys

* Iterate keys
with given prefix

* Track changes
on keys

Description of
most keys created by corosync itself can be found in
cmap_keys (8).
