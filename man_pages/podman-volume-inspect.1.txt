NAME

podman-volume-inspect
- Get detailed information on one or more volumes

SYNOPSIS

podman
volume inspect [options] volume [...]

DESCRIPTION

Display
detailed information on one or more volumes. The output can
be formatted using the --format flag and a Go
template. To get detailed information about all the existing
volumes, use the --all flag. Volumes can be queried
individually by providing their full name or a unique
partial name.

OPTIONS

--all,
-a 
Inspect all volumes.

--format,
-f=format 
Format volume output using Go template

Valid
placeholders for the Go template are listed below:

--help 
Print usage statement
