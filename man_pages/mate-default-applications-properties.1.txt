NAME
       mate-default-applications-properties — is a control center applet.

SYNOPSIS
       mate-default-applications-properties [OPTIONS]...

DESCRIPTION
       MATE Default Applications Properties is a control center applet to edit
       default applications.

       It allows one to set default file manager, email client,  web  browser,
       etc.

OPTIONS
       -p,  --show-page=PAGE
                 Specify  the  name  of  the  page  to show (internet|multime‐
                 dia|system|a11y)
