NAME
       Image::ExifTool::JVC - JVC EXIF maker notes tags

SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.

DESCRIPTION
       This module contains routines used by Image::ExifTool to interpret JVC
       maker notes.
