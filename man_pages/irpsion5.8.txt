SYNOPSIS

irpsion5
[ -s file ] [ -r ] [ -b ] [ -v ]
[ -q ] [ -S ms ] [ -D n ]

OPTIONS

-s file
send file to the Psion

-r
receive a file from the Psion

-b
receive multiple files (batch mode)

-v
verbose mode, progress indication

-q quiet
mode, no progress indication

-S ms
set discovery slot timeout, standard 80ms, default 120ms

-D n
debug level

DESCRIPTION

N.N.

irpsion5
must be run as root or installed setuid root, as it requires
root privileges.
