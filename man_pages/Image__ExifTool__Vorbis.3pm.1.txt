NAME
       Image::ExifTool::Vorbis - Read Ogg Vorbis audio meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information from Ogg Vorbis audio headers.
