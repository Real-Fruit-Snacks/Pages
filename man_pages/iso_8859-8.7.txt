NAME

iso_8859-8 -
ISO 8859-8 character set encoded in octal, decimal, and
hexadecimal

DESCRIPTION

The ISO 8859
standard includes several 8-bit extensions to the ASCII
character set (also known as ISO 646-IRV). ISO 8859-8
encodes the characters used in Modern Hebrew.

NOTES

ISO 8859-8 was
also known as ISO-IR-138. ISO 8859-8 includes neither short
vowels nor diacritical marks, and Yiddish is not provided
for.
