NAME

security_check_context
- check the validity of a SELinux context

SYNOPSIS

#include

int
security_check_context(const char
*con);

int
security_check_context_raw(const char
*con);

DESCRIPTION

security_check_context()
returns 0 if SELinux is running and the context is valid,
otherwise it returns -1.

security_check_context_raw()
behaves identically to security_check_context() but
does not perform context translation.
