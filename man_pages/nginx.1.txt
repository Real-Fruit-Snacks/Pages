NAME

nginx −
small, but very powerful and efficient web server

SYNOPSIS

nginx
[options] 

DESCRIPTION

nginx is
a server that can be used as standalone HTTP server and as a
reverse proxy server before some Apache or another big
server to reduce load to backend servers by many concurrent
HTTP−sessions.

OPTIONS

A summary of
options is included below:

−c
 Specifies a particular
configuration file for nginx to load. 
−v Show version of program. 
−t Tests nginx configuration

You can find
further information about this webserver in
http://nginx.net/docs_en.txt.
