NAME

uuid_compare -
compare whether two UUIDs are the same

SYNOPSIS

#include

int
uuid_compare(const uuid_t uu1, const
uuid_t uu2)

DESCRIPTION

The
uuid_compare() function compares the two supplied
uuid variables uu1 and uu2 to each other.

RETURN VALUE

Returns an
integer less than, equal to, or greater than zero if
uu1 is found, respectively, to be lexicographically
less than, equal, or greater than uu2.
