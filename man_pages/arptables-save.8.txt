NAME

arptables-save
&mdash; dump arptables rules to stdout

SYNOPSIS

arptables-save

DESCRIPTION

arptables-save
is used to dump the contents of an ARP Table in easily
parseable format to STDOUT. Use I/O-redirection provided by
your shell to write to a file.
