NAME

ifmcstat &mdash; dump
multicast group management statistics per interface

SYNOPSIS

ifmcstat

DESCRIPTION

The ifmcstat utility
dumps multicast group information from the kernel. It is
similar but not identical to the output from netstat
&minus;ina.

There are no
command line options.

Only root can
use ifmcstat.
