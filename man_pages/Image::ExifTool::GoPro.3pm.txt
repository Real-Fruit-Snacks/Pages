NAME
       Image::ExifTool::GoPro - Read information from GoPro videos

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to decode
       metadata from GoPro MP4 videos.
