NAME

gh-project-view
- View a project

SYNOPSIS

gh project
view [] [flags]

OPTIONS

--format

Output format: {json}

-q, --jq

Filter JSON output using a jq
expression

--owner

Login of the owner. Use
"@me" for the current user.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

-w, --web

Open a project in the
browser

EXAMPLE

# view the
current user&rsquo;s project "1" 
gh project view 1

# open user
monalisa&rsquo;s project "1" in the browser 
gh project view 1 --owner monalisa --web
