NAME

iso_8859-5 -
ISO 8859-5 character set encoded in octal, decimal, and
hexadecimal

DESCRIPTION

The ISO 8859
standard includes several 8-bit extensions to the ASCII
character set (also known as ISO 646-IRV). ISO 8859-5
encodes the Cyrillic characters used in many East European
languages.
