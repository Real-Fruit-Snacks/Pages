NOME
       ppdcfile  - formato de ficheiro fonte do compilador de ppd do cups (de‐
       scontinuado)

DESCRIÇÃO
       O compilador de PPD do CUPS lê meta-ficheiros que contém descrições  de
       um  ou  mais  ficheiros  PPD  a serem gerados pelo ppdc(1). Este manual
       fornece uma referência rápida às palavras chave suportadas e  deve  ser
       usado em conjunto com a ajuda online para o CUPS.

       O  formato do ficheiro fonte é texto simples ASCII que pode ser editado
       usando o seu editor de texto favorito. Comentários são  suportados  us‐
       ando os mecanismos de comentário C (/* ... */) e C++ (// ...).

       A informação de driver de impressora pode ser agrupada e partilhada us‐
       ando chavetas ({ ... }); ficheiros PPD são escritos quando é vista  uma
       chaveta  de  fecho  ou o fim-do-ficheiro e uma directiva PCFileName foi
       definida.

       As directivas podem ser colocadas em qualquer sítio numa  linha  e  são
       seguidas  por um ou mais valores. O seguinte é uma lista das directivas
       disponíveis e dos valores que elas aceitam.

       #define name value

       #elif {name | value}

       #else

       #endif

       #font name encoding "version" charset status

       #if {name | value}

       #include <filename>

       #include "filename"

       #media name width length

       #media "name/text" width length

       #po locale "filename"

       Attribute name "" value

       Attribute name keyword value

       Attribute name "keyword/text" value

       Choice name "code"

       Choice "name/text" "code"

       ColorDevice boolean-value

       ColorModel name colorspace colororder compression

       ColorModel "name/text" colorspace colororder compression

       ColorProfile resolution/mediatype gamma density matrix

       Copyright "text"

       CustomMedia name width length left bottom right  top  "size-code"  "re‐
       gion-code"

       CustomMedia  "name/text" width length left bottom right top "size-code"
       "region-code"

       Cutter boolean-value

       Darkness temperature name

       Darkness temperature "name/text"

       DriverType type

       Duplex type

       Filter mime-type cost program

       Finishing name

       Finishing "name/text"

       Font *

       Font name encoding "version" charset status

       Group name

       Group "name/text"

       HWMargins left bottom right top

       InputSlot position name

       InputSlot position "name/text"

       Installable name

       Installable "name/text"

       LocAttribute name "keyword/text" value

       ManualCopies boolean-value

       Manufacturer "name"

       MaxSize width length

       MediaSize name

       MediaType type name

       MediaType type "name/text"

       MinSize width length

       ModelName "name"

       ModelNumber number

       Option name type section order

       Option "name/text" type section order

       PCFileName "filename.ppd"

       Resolution colorspace bits-per-color row-count row-feed row-step name

       Resolution  colorspace  bits-per-color  row-count   row-feed   row-step
       "name/text"

       SimpleColorProfile resolution/mediatype density yellow-density red-den‐
       sity gamma red-adjust green-adjust blue-adjust

       Throughput pages-per-minute

       UIConstraints "*Option1 *Option2"

       UIConstraints "*Option1 Choice1 *Option2"

       UIConstraints "*Option1 *Option2 Choice2"

       UIConstraints "*Option1 Choice1 *Option2 Choice2"

       VariablePaperSize boolean-value

       Version number

NOTAS
       Ficheiros PPD estão descontinuados e não irão ser mais  suportados  num
       futuro  lançamento  do CUPS. Impressoras que não suportem IPP podem ser
       suportadas usando aplicações como a ippeveprinter(1).
