NAME
       pbmtosunicon - convert a PBM image into a Sun icon

SYNOPSIS
       pbmtosunicon [pbmfile]

DESCRIPTION
       This program is part of Netpbm(1).

       pbmtosunicon reads a PBM image as input and produces a Sun icon as out-
       put.

OPTIONS
       There are no command line options defined specifically  for  pbmtosuni-
       con, but it recognizes the options common to all programs based on lib-
       netpbm (See
        Common Options <index.html#commonoptions> .)
