NAME

votequorum_setexpected
- Sets the expected votes for the cluster

SYNOPSIS

#include

int
votequorum_setexpected(votequorum_handle_t
handle, int expected_votes);

DESCRIPTION

The
votequorum_setexpected function is used to change the
expected votes in the cluster. Expected votes is used to
calculate quorum and should normally be the total number of
votes that will exist when all the expected nodes are
joined. Quorum will usually be half of this (rounded up). It
is not possible to set expected votes up so that it makes
the cluster inquorate using this command.

RETURN VALUE

This call
returns the CS_OK value if successful, otherwise an error is
returned.
