NAME

apm &minus;
interface with the APM subsystem

SYNOPSIS

apm [
&minus;VvmMsSdin ]

DESCRIPTION

This command
requires the Advanced Power Management subsystem. If run
without arguments it reads /proc/apm and presents the
output in a human-readable format.

Given a
&minus;S or &minus;s option it requests that
the machine be put into the APM standby or the APM suspend
state, respectively. For a brief description of these
states, see apmd(8).

OPTIONS

&minus;V,
&minus;&minus;version

Print the version of the
program and exit.

&minus;v,
&minus;&minus;verbose

Also print version information
for the APM subsystem. The APM subsystem consists of
power-management hardware, firmware usually referred to as
the APM BIOS and a driver in the operating system
kernel.

&minus;m,
&minus;&minus;minutes

Print the time remaining as a
number of minutes. The default is to print the time
remaining in "hours:minutes" format.

&minus;M,
&minus;&minus;monitor

Continuously monitor and update
the status information. Assumes that the standard output is
a terminal.

&minus;S,
&minus;&minus;standby

Request that the machine be put
into the APM standby state.

&minus;s,
&minus;&minus;suspend

Request that the machine be put
into the APM suspend state.

&minus;d,
&minus;&minus;debug

Print APM status information in
a format more useful for debugging. Implies &minus;v.

FILES

/proc/apm
