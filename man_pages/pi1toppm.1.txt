NAME
       pi1toppm - convert an Atari Degas .pi1 into a PPM image

SYNOPSIS
       pi1toppm

       [pi1file]

DESCRIPTION
       This program is part of Netpbm(1).

       pi1toppm reads an Atari Degas .pi1 file as input and produces a PPM im‐
       age as output.

OPTIONS
       There are no command line options defined  specifically  for  pi1toppm,
       but it recognizes the options common to all programs based on libnetpbm
       (See
        Common Options ⟨index.html#commonoptions⟩ .)
