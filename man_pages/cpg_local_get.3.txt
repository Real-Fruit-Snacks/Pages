NAME

cpg_local_get -
Returns the local node id

SYNOPSIS

#include

int
cpg_local_get(cpg_handle_t handle, unsigned
int *local_nodeid);

DESCRIPTION

The
cpg_local_get function is used to determine the local
node&rsquo;s identifier. The argument handle is used
to reference the cpg instantiation. The argument
local_nodeid will return the 32 bit node id.

ERRORS

The errors are
undocumented.
