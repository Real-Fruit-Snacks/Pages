NAME

memmem - search
for memory block inside another memory block

SYNOPSIS

#include

void *memmem(const void *v, size_t
size, 
const void *pat, size_t patsize);

DESCRIPTION

memmem
searches for the pattern pat (length patsize
bytes) in the memory block v (length size
bytes).

RETURN VALUE

memmem
returns a pointer to the first byte of the first occurrence
it finds, or NULL if it doesnâ€™t find any
occurrence.
