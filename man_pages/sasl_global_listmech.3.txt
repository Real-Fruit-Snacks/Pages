NAME

sasl_global_listmech
- Cyrus SASL documentation

SYNOPSIS

#include

const char **
sasl_global_listmech();

DESCRIPTION

const char
**sasl_global_listmech();

sasl_global_listmech
returns a null-terminated array of strings that lists all
mechanisms that are loaded by either the client or server
side of the library.

RETURN VALUE

Returns a
pointer to the array on success. NULL on failure (sasl
library uninitialized).
