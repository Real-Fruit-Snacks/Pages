NAME
       clock_t - system time

LIBRARY
       Standard C library (libc)

SYNOPSIS
       #include <time.h>

       typedef /* ... */ clock_t;

DESCRIPTION
       Used  for  system  time  in  clock  ticks or CLOCKS_PER_SEC (defined in
       <time.h>).  According to POSIX, it is an integer type or a  real-float‚Äê
       ing type.

STANDARDS
       C99 and later; POSIX.1-2001 and later.

NOTES
       The  following  headers  also  provide  this  type:  <sys/types.h>  and
       <sys/time.h>.

SEE ALSO
       times(2), clock(3)
