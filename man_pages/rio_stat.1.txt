NAME

rio_stat
&minus; display statistics for Diamond Rio 500 digital audio
player

SYNOPSIS

rio_stat
[options]

DESCRIPTION

The
rio_stat command displays a variety of information
concerning the Rio, such as

*

Firmware version

*

Total and available memory

*

Folders which have been created

*

Songs which have been uploaded

OPTIONS

&minus;h,
&minus;&minus;help

Display a short usage
summary.

&minus;t,
&minus;&minus;terse

Display information in an
abbreviated format.

&minus;v,
&minus;&minus;version

Display program version.
