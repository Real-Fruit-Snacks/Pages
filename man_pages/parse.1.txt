create a parse tree of the input

SYNOPSIS

parse
GRAMMAR PDA [OP]

DESCRIPTION

Parses the text
on standard input using the specified GRAMMAR and prints the
parse tree to standard output. PDA must be the push-down
automaton for the context free GRAMMAR. OP is an optional
operator-precedence table.
