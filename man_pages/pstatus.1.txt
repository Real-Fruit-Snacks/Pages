NAME

pstatus &minus;
show device status

SYNOPSIS

pstatus
[&minus;d LEVEL] [&minus;h]

DESCRIPTION

pstatus
displays information about a device connected through
SynCE.

OPTIONS

-d
LEVEL

Set debug log level:

0 - No logging
(default)

1 - Errors
only

2 - Errors and
warnings

3 -
Everything

-h

Display help message.
