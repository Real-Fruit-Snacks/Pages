NAME

gh-project-edit
- Edit a project

SYNOPSIS

gh project
edit [] [flags]

OPTIONS

-d,
--description 

New description of the
project

--format

Output format: {json}

-q, --jq

Filter JSON output using a jq
expression

--owner

Login of the owner. Use
"@me" for the current user.

--readme

New readme for the project

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

--title

New title for the project

--visibility

Change project visibility:
{PUBLIC|PRIVATE}

EXAMPLE

# edit the
title of monalisa&rsquo;s project "1" 
gh project edit 1 --owner monalisa --title "New
title"
