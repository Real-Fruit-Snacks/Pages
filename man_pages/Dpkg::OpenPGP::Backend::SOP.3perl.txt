NAME
       Dpkg::OpenPGP::Backend::SOP - OpenPGP backend for SOP

DESCRIPTION
       This module provides a class that implements the OpenPGP backend for
       the Stateless OpenPGP Command-Line Interface, as described in
       <https://datatracker.ietf.org/doc/draft-dkg-openpgp-stateless-cli>.

       Note: This is a private module, its API can change at any time.
