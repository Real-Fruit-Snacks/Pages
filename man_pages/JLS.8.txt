NAME

jls &mdash; list active
jails

SYNOPSIS

jls

DESCRIPTION

The jls utility lists all
active jails. Each jail is represented by one row which
contains the following columns: jail identifier (JID), IP
address, hostname, and path.

SEE ALSO

jail(2), jail(8), jexec(8)

HISTORY

The jls utility was added
in FreeBSD 5.1.
