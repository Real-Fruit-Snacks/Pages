NAME
       Image::ExifTool::Olympus - Olympus/Epson maker notes tags

SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.

DESCRIPTION
       This module contains definitions required by Image::ExifTool to
       interpret Olympus or Epson maker notes in EXIF information.
