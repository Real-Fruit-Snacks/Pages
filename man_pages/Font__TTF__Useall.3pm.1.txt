NAME
       Font::TTF::Useall - shortcut to 'use' all the Font::TTF modules

SYNOPSIS
           use Font::TTF::Useall;

DESCRIPTION
       Useful for debugging, this module simply does a 'use' on all the other
       modules that are part of Font::TTF.
