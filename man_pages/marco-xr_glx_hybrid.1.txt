NAME
       marco-xr_glx_hybrid - Marco window manager wrapper

SYNOPSIS
       marco-xr_glx_hybrid

DESCRIPTION
       A  wrapper  for  the  Marco window manager that invokes picom to enable
       xr_glx_hybrid compositing.
