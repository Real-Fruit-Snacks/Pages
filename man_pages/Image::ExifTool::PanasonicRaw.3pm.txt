NAME
       Image::ExifTool::PanasonicRaw - Read/write Panasonic/Leica RAW/RW2/RWL
       meta information

SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.

DESCRIPTION
       This module contains definitions required by Image::ExifTool to read
       and write meta information in Panasonic/Leica RAW, RW2 and RWL images.
