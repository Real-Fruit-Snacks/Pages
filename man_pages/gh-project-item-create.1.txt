NAME

gh-project-item-create
- Create a draft issue item in a project

SYNOPSIS

gh project
item-create [] [flags]

OPTIONS

--body

Body for the draft issue

--format

Output format: {json}

-q, --jq

Filter JSON output using a jq
expression

--owner

Login of the owner. Use
"@me" for the current user.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

--title

Title for the draft issue

EXAMPLE

# create a
draft issue in the current user&rsquo;s project
"1" 
gh project item-create 1 --owner "@me" --title
"new item" --body "new item body"
