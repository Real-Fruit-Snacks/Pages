NAME

mate-screensaver
- The MATE Desktop Screensaver and Locker

SYNOPSIS

mate-screensaver
[OPTIONS]

DESCRIPTION

MATE
Screensaver is the default screen saver and locker for the
MATE Desktop. It is designed with simplicity and security in
mind. mate-screensaver integrates well with the MATE
desktop, and it supports User Switching and the ability to
lock down configuration settings.
