NAME

nbd_set_uri_allow_local_file
- set the allowed transports in NBD URIs

SYNOPSIS

#include

int nbd_set_uri_allow_local_file ( 
struct nbd_handle *h, bool allow 
);

DESCRIPTION

Allow NBD URIs
to reference local files. This is disabled by
default.

Currently this
setting only controls whether the "tls-psk-file"
parameter in NBD URIs is allowed.

RETURN VALUE

If the call is
successful the function returns 0.

ERRORS

On error -1 is
returned.

Refer to
"ERROR HANDLING" in libnbd(3) for how to
get further details of the error.

The following
parameters must not be NULL: "h". For more
information see "Non-NULL parameters" in
libnbd(3).
