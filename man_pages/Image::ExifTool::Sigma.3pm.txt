NAME
       Image::ExifTool::Sigma - Sigma/Foveon EXIF maker notes tags

SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.

DESCRIPTION
       This module contains definitions required by Image::ExifTool to
       interpret Sigma and Foveon maker notes in EXIF information.
