NAME

nbd_get_opt_mode
- return whether option mode was enabled

SYNOPSIS

#include

int nbd_get_opt_mode ( 
struct nbd_handle *h 
);

DESCRIPTION

Return true if
option negotiation mode was enabled on this handle.

RETURN VALUE

This call
returns a boolean value.

ERRORS

This function
does not fail.

The following
parameters must not be NULL: "h". For more
information see "Non-NULL parameters" in
libnbd(3).

VERSION

This function
first appeared in libnbd 1.4.

If you need to
test if this function is available at compile time check if
the following macro is defined:

#define
LIBNBD_HAVE_NBD_GET_OPT_MODE 1

SEE ALSO

nbd_create(3),
nbd_set_opt_mode(3), libnbd(3).
