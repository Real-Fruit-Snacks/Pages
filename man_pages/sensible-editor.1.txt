NOME
       sensible-editor - edição sensible

RESUMO
       sensible-editor [OPÇÕES...]

DESCRIÇÃO
       sensible-editor faz decisões sensible sobre qual editor chamar. Progra‐
       mas em Debian podem usar este script como  o  seu  editor  predefinido.
       sensible-editor tenta fazer na seguinte ordem:

       1      Se  a  variável  de  ambiente  VISUAL  existir,  executa  VISUAL
              [OPÇÔES]

       2      Se  a  variável  de  ambiente  EDITOR  existir,   executa   EDI‐
              TOR [OPÇÕES]

       3      sonda o conteúdo do ficheiro ~/.selected_editor e, se a variável
              de  ambiente  SELECTED_EDITOR  existir   executa   SELECTED_EDI‐
              TOR [OPÇÕES]

       4      corre o comando editor [OPÇÕES]

       5      finalmente corre o comando nano-tiny [OPÇÕES]

VEJA TAMBÉM
       environ(7) para documentação das variáveis do EDITOR, VISUAL
       select-editor(1)  para mudar o editor predefinido de um utilizador.
       editor(1) para o editor predefinido geral do sistema.

BUGS
       Este  programa  está protegido contra bomba fork trivial, quando o uti‐
       lizador define EDITOR=sensible-editor são ainda  possíveis  loops  mais
       largos.

STANDARD
       A documentação do comportamento do sensible-utils sob um sistema debian
       está disponível sob a secção 11.4 da debian-policy normalmente  instal‐
       ada em /usr/share/doc/debian-policy (você pode precisar de instalar de‐
       bian-policy)
