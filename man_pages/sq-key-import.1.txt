NAME

sq key import -
Import keys into the key store

SYNOPSIS

sq key
import [OPTIONS] KEY_FILE

DESCRIPTION

Import keys
into the key store.

OPTIONS

Subcommand
options 
KEY_FILE

Import the keys in KEY_FILE

Global
options 
See sq(1) for a description of the global
options.

EXAMPLES

Import the keys
into the keystore server.

sq key import
alice-secret.pgp

sq-key(1).

documentation see https://book.sequoia-pgp.org&gt;.
