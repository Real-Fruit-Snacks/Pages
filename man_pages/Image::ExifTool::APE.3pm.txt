NAME
       Image::ExifTool::APE - Read Monkey's Audio meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information from Monkey's Audio (APE) audio files.

BUGS
       Currently doesn't parse MAC header unless it is at the start of the
       file.
