NAMN
       deb-postinst - paketutvecklarskript som körs före installation

SYNOPS
       DEBIAN/preinst

BESKRIVNING
       Ett paket kan utföra flera åtgärder före installation via
       utvecklarskript, genom att ta med en körbar fil preinst i sitt
       styrarkiv (dvs. DEBIAN/preinst när paketet skapas).

       Skriptet kan anropas på följande sätt:

       ny-preinst install
           Innan paketet installeras.

       ny-preinst install gammal-version ny-version
           Innan ett borttaget paketet uppgraderas.

           ny-version sänds endast sedan dpkg 1.18.5.

       ny-preinst upgrade gammal-version ny-version
           Innan paketet uppgaderas.

           ny-version sänds endast sedan dpkg 1.18.5.

       gammal-preinst abort-upgrade ny-version
           Om postrm misslyckas vid uppgradering eller misslyckas vid
           misslyckad uppgradering.
