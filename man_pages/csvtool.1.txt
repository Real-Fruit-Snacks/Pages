NAME

csvtool - tool
for performing manipulations on CSV files from shell
scripts

SYNOPSIS

csvtool [options]
command [command-args] input.csv...

DESCRIPTION

This manual
page documents briefly the csvtool command.

This manual
page was written for the Debian distribution because the
original program does not have a manual page. Instead, it
has documentation accessible with the --help
option.

csvtool
is a tool for performing manipulations on CSV files from
shell scripts.

OPTIONS

-help
--help

Show help.
