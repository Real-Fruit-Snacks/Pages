NAME
       Image::ExifTool::Parrot - Read timed metadata from Parrot drone videos

SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.

DESCRIPTION
       This module contains definitions required by Image::ExifTool to
       interpret timed metadata from the 'mett' frame found in Parrot drone
       MP4 videos.
