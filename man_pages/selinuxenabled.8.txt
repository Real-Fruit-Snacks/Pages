NAME

selinuxenabled
- tool to be used within shell scripts to determine if
selinux is enabled

SYNOPSIS

selinuxenabled

DESCRIPTION

Indicates
whether SELinux is enabled or disabled.

EXIT STATUS

It exits with
status 0 if SELinux is enabled and 1 if it is not
enabled.
