NAME
       Image::ExifTool::GPS - EXIF GPS meta information tags

SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.

DESCRIPTION
       This module contains definitions required by Image::ExifTool to
       interpret GPS (Global Positioning System) meta information in EXIF
       data.
