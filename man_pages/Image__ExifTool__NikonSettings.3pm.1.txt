NAME
       Image::ExifTool::NikonSettings - Read Nikon user settings

SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.

DESCRIPTION
       This module contains the definitions necessary to read the user
       settings for Nikon cameras such as the D6 and Z7mk2.
