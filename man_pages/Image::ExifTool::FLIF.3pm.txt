NAME
       Image::ExifTool::FLIF - Read/write FLIF meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains routines required by Image::ExifTool to read and
       write meta information in FLIF (Free Lossless Image Format) images.
