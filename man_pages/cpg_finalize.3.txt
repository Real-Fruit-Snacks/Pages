NAME

cpg_finalize -
Terminate a connection to the CPG service

SYNOPSIS

#include

int
cpg_finalize(cpg_handle_t handle);

DESCRIPTION

The
cpg_finalize function is used to close a connection
to the closed process group API. Once the connection is
finalized, the handle may not be used again by applications.
No more callbacks will be dispatched from the
cpg_dispatch function.

RETURN VALUE

This call
returns the CS_OK value if successful, otherwise an error is
returned.

ERRORS

The errors are
undocumented.
