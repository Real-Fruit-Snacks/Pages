NAME
       Image::ExifTool::Rawzor - Read meta information from Rawzor compressed
       images

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information from Rawzor compressed images.
