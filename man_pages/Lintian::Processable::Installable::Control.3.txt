NAME
       Lintian::Processable::Installable::Control - access to collected
       control file data

SYNOPSIS
           use Lintian::Processable;

DESCRIPTION
       Lintian::Processable::Installable::Control provides an interface to
       control file data.

INSTANCE METHODS
       control
           Returns the index for a binary control file.

AUTHOR

       Lintian.
