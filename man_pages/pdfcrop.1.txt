NAME

PDFCROP &minus;
crop pdf files to their minimal size.

SYNOPSIS

pdfcrop
[OPTIONS ...] input[.pdf] [output file]

DESCRIPTION

pdfcrop
is a utility to calculate and remove empty margins from each
page in the input PDF file. The resulting output file
occupies the minimal paper size needed for the contents and
is therefore suitable for inclusion as a graphic

OPTIONS

&minus;&minus;help

print usage

&minus;&minus;version

print version

&minus;&minus;(no)verbose

verbose printing (default:
false)

&minus;&minus;(no)debug

debug informations (default:
false)

&minus;&minus;gscmd

call of ghostscript (default:
gs)

&minus;&minus;pdftexcmd

call of pdfTeX (default:
pdftex)

&minus;&minus;margins
"  
"

(0 0 0 0) add extra margins,
unit is bp. If only one number is given, then it is used for
all margins, in the case of two numbers they are also used
for right and bottom.

&minus;&minus;(no)clip

clipping support, if margins
are set (default: false)

&minus;&minus;(no)hires

using
&rsquo;%%HiResBoundingBox&rsquo; instead of
&rsquo;%%BoundingBox&rsquo; (default: false)

&minus;&minus;papersize

parameter for gs&rsquo;s
&minus;sPAPERSIZE=, use only with older gs
versions 

EXAMPLES

pdfcrop
&minus;&minus;margins 10 input.pdf output.pdf

pdfcrop
&minus;&minus;margins &rsquo;5 10 5 20&rsquo;
&minus;&minus;clip input.pdf output.pdf
