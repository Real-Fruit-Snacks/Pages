NAME

runc-events
- display container events and statistics.

SYNOPSIS

runc
events [option ...] container-id

DESCRIPTION

The
events command displays information about the
container. By default, it works continuously, displaying
stats every 5 seconds, and container events as they
occur.

OPTIONS

--interval
time

Set the stats collection
interval. Default is 5s.

--stats

Show the containerâ€™s
stats once then exit.
