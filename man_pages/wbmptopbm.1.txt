NAME

wbmptopbm -
convert a wireless bitmap (wbmp) file to a PBM

SYNOPSIS

wbmptopbm

[wbmpfile]

DESCRIPTION

This program is
part of Netpbm(1).

wbmptopbm
reads a wbmp file as input and produces a PBM image as
output.

OPTIONS

There are no
command line options defined specifically for
wbmptopbm, but it recognizes the options common to
all programs based on libnetpbm (See
 Common Options
.)

LIMITATIONS

wbmptopbm
recognizes only WBMP type 0. This is the only type specified
in the WAP 1.1 specifications.
