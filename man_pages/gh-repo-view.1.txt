NAME

gh-repo-view -
View a repository

SYNOPSIS

gh repo view
[] [flags]

DESCRIPTION

Display the
description and the README of a GitHub repository.

With no
argument, the repository for the current directory is
displayed.

With
’--web’, open the repository in a web browser
instead.

With
’--branch’, view a specific branch of the
repository.

OPTIONS

-b,
--branch 

View a specific branch of the
repository

-q, --jq

Filter JSON output using a jq
expression

--json

Output JSON with the specified
fields

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

-w, --web

Open a repository in the
browser
