NAME
       gh-repo-archive - Archive a repository

SYNOPSIS
       gh repo archive [<repository>] [flags]

DESCRIPTION
       Archive a GitHub repository.

       With no argument, archives the current repository.

OPTIONS
       -y, --yes
              Skip the confirmation prompt

SEE ALSO
       gh-repo(1)
