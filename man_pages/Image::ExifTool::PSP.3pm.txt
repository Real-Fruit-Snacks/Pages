NAME
       Image::ExifTool::PSP - Read Paint Shop Pro meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains routines required by Image::ExifTool to extract
       information from Paint Shop Pro images.
