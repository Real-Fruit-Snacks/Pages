NAME

jexec — execute a
command inside an existing jail

SYNOPSIS

jexec jid command
...

DESCRIPTION

The jexec utility
executes command inside the jail identified by
jid.

SEE ALSO

jail_attach(2), jail(8),
jls(8)

HISTORY

The jexec utility was
added in FreeBSD 5.1.
