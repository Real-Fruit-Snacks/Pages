NAME
       xbmtopbm - convert an X11 or X10 bitmap to a PBM image

SYNOPSIS
       xbmtopbm

       [bitmapfile]

DESCRIPTION
       This program is part of Netpbm(1).

       xbmtopbm  reads  an X11 or X10 bitmap as input and produces a PBM image
       as output.

OPTIONS
       There are no command line options defined  specifically  for  xbmtopbm,
       but it recognizes the options common to all programs based on libnetpbm
       (See
        Common Options ⟨index.html#commonoptions⟩ .)
