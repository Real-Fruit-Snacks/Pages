NAME

sasl_getsimple_t
- Cyrus SASL documentation

SYNOPSIS

#include

int
sasl_getsimple_t(void *context, 
int id, 
const char ** result, 
unsigned * len);

DESCRIPTION

int
sasl_getsimple_t(void *context, 
int id, 
const char ** result, 
unsigned * len);

sasl_getsimple_t is used
to retrieve simple things from the application. In practice
this is authentication name, authorization name, and realm.

Parameters

&bull;

context &ndash; SASL connection context

&bull;

id &ndash;

indicates which
value is being requested. Possible values include:

&bull;

SASL_CB_USER - Client user
identity to login as

&bull;

SASL_CB_AUTHNAME - Client authentication name

&bull;

SASL_CB_LANGUAGE - Comma-separated list of RFC
1766 languages

&bull;

SASL_CB_CNONCE - Client-nonce (for testing mostly)

&bull;

result
&ndash; value of the item requested

&bull;

len &ndash; lenth of the
result 

RETURN VALUE

SASL callback
functions should return SASL return codes. See sasl.h for a
complete list. SASL_OK indicates success.
