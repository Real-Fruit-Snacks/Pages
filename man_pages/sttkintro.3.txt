NAME

sttk_intro -
introduction to the ShapeTools Toolkit Library

SYNOPSIS

#include

#include 

char* stVersion
(void);

DESCRIPTION

The ShapeTools
Toolkit library (sttk) is a set of functions to be used in
various parts of the ShapeTools configuration management
toolkit.

Inclusion of
config.h may be omitted when atfs.h is
included.

stVersion
returns a version identification string for the ShapeTools
Toolkit library.
