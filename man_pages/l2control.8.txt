L2CONTROL(8)

NAME

l2control &mdash; L2CAP
configuration utility

SYNOPSIS

l2control
[&minus;hn] &minus;a BD_ADDR command
[parameters ...]

DESCRIPTION

The l2control utility
connects to the local device with the specified BD_ADDR and
attempts to send the specified command. The l2control
utility will print results to the standard output and error
messages to the standard error.

The options are
as follows:

&minus;a
BD_ADDR

Connect to the local device
with the specified BD_ADDR. Example: &minus;a
00:01:02:03:04:05.

&minus;h

Display usage
message and exit.

&minus;n

Show Bluetooth
addresses as numbers. Normally l2control attempts to
resolve Bluetooth addresses, and display them
symbolically.

command

One of the supported commands
(see below). Special command help can be used to
obtain the list of all supported commands. To get more
information about specific command use help
command.

parameters

One or more optional space
separated command parameters.

COMMANDS

The currently supported node
commands in l2control are:

Read_Node_Flags 
Read_Debug_Level 
Write_Debug_Level 
Read_Connection_List 
Read_Channel_List 
Read_Auto_Disconnect_Timeout 
Write_Auto_Disconnect_Timeout

DIAGNOSTICS

The l2control utility
exits 0 on success, and >0 if an error
occurs.
