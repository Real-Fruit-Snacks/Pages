NAME
       mtvtoppm - convert output from an MTV or PRT ray tracer into a PPM

SYNOPSIS
       mtvtoppm [mtvfile]

DESCRIPTION
       This program is part of Netpbm(1).

       mtvtoppm  reads an input file from Mark VanDeWettering's MTV ray tracer
       and produces a PPM image as output.

       The PRT raytracer also produces this format.

OPTIONS
       There are no command line options defined  specifically  for  mtvtoppm,
       but it recognizes the options common to all programs based on libnetpbm
       (See
        Common Options ⟨index.html#commonoptions⟩ .)
