NAME
       mate-menu - Advanced menu for the MATE desktop environment's panel

SYNOPSIS
       mate-menu [option]

DESCRIPTION
       An  advanced menu for MATE. Supports filtering, favorites, autosession,
       and many other features.

OPTIONS
       [--]help, [-]h, [-]?
              Display command line options.
