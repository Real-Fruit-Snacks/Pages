NAME

shell &minus;
Print the message of the day and launch a shell

DESCRIPTION

shell is
a simple script that prints the message of the day
(/etc/motd) and launches a shell.

$SHELL will be
invoked if it exists. Otherwise, /bin/sh will be used.

http://launchpad.net/byobu
