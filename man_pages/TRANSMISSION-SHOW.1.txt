NAME

transmission-show —
command-line utility to show .torrent file metadata

SYNOPSIS

transmission-show
[-h] [-m] [-s] [torrentfile]

DESCRIPTION

transmission-show shows
BitTorrent .torrent file metadata

OPTIONS 
-h --help

Show a short help page and
exit.

-m --magnet

Show a magnet link for the
specified .torrent file

-s --scrape

Ask the torrent’s
trackers how many peers are in the torrent’s swarm
