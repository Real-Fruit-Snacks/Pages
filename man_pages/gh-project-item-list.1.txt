NAME

gh-project-item-list
- List the items in a project

SYNOPSIS

gh project
item-list [] [flags]

OPTIONS

--format

Output format: {json}

-q, --jq

Filter JSON output using a jq
expression

-L, --limit
 (default 30)

Maximum number of items to
fetch

--owner

Login of the owner. Use
"@me" for the current user.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

EXAMPLE

# list the
items in the current users&rsquo;s project "1"

gh project item-list 1 --owner "@me"
