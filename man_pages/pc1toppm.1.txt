NAME
       pc1toppm - convert an Atari Degas .pc1 into a PPM image

SYNOPSIS
       pc1toppm

       [pc1file]

DESCRIPTION
       This program is part of Netpbm(1).

       pc1toppm reads an Atari Degas .pc1 file as input and produces a PPM im‐
       age as output.

       The .pc1 format is a compressed (run length encoded) variation on .pi1.

OPTIONS
       There are no command line options defined  specifically  for  pc1toppm,
       but it recognizes the options common to all programs based on libnetpbm
       (See
        Common Options ⟨index.html#commonoptions⟩ .)

HISTORY
       pc1toppm was new in Netpbm 10.22 (April 2004)
