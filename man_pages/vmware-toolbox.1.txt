NAME

vmware−toolbox
− GUI toolbox

SYNOPSIS

vmware−toolbox

DESCRIPTION

This is a
per−user process that, like
vmware−user(1), must be run in an X11 session,
and isn’t needed otherwise. Also like
vmware−user(1), it does not support multiple
concurrent users, and depends on certain Gtk shared objects
at runtime.

OPTIONS

vmware−toolbox
has no options.
