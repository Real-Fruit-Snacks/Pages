NAME
       getppid — get the parent process ID

SYNOPSIS
       #include <unistd.h>

       pid_t getppid(void);

DESCRIPTION
       The getppid() function shall return the parent process ID of the  call‐
       ing process.

RETURN VALUE
       The  getppid()  function shall always be successful and no return value
       is reserved to indicate an error.

ERRORS
       No errors are defined.
