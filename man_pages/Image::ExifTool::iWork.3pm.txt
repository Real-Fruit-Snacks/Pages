NAME
       Image::ExifTool::iWork - Read Apple iWork '09 XML+ZIP files

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information from Apple iWork '09 XML+ZIP files.
