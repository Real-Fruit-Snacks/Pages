OHCI(4) BSD Kernel Interfaces Manual OHCI(4)

NAME

ohci &mdash; OHCI USB
Host Controller driver

SYNOPSIS

device ohci

DESCRIPTION

The ohci driver provides
support for OHCI-type PCI based USB controllers.

HARDWARE

The ohci driver supports
all OHCI v1.0 compliant controllers including:

&bull;

AcerLabs M5237 (Aladdin-V)

&bull;

AMD-756

&bull;

OPTi 82C861 (FireLink)

&bull;

NEC uPD 9210

&bull;

CMD Tech 670 (USB0670)

&bull;

CMD Tech 673 (USB0673)

&bull;

NVIDIA nForce3

&bull;

Sun PCIO-2 (RIO USB)

SYSCTL VARIABLES

The following variables are
available as both sysctl(8) variables and loader(8)
tunables:

hw.usb.ohci.debug

Debug output level, where 0 is
debugging disabled and larger values increase debug message
verbosity. Default is 0.

SEE ALSO

ehci(4), uhci(4), xhci(4)

HISTORY

The ohci device driver
first appeared in FreeBSD 3.0.
