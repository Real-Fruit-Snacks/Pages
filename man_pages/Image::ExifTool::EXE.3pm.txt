NAME
       Image::ExifTool::EXE - Read executable file meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information from various types of Windows, MacOS and Unix
       executable and library files.
