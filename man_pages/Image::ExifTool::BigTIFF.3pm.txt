NAME
       Image::ExifTool::BigTIFF - Read Big TIFF meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains routines required by Image::ExifTool to read meta
       information in BigTIFF images.
