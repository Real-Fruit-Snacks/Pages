NAME

dacf.conf
&minus; device auto-configuration configuration file

SYNOPSIS

/etc/dacf.conf

DESCRIPTION

The kernel uses
the dacf.conf file to automatically configure hot
plugged devices. Because the dacf.conf file contains
important kernel state information, it should not be
modified.

The format of
the /etc/dacf.conf file is not public and might
change in versions of the Solaris operating environment that
are not compatible with Solaris 8.
