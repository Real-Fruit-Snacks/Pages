NAME
       Image::ExifTool::OOXML - Read Office Open XML+ZIP files

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information from Office Open XML files.  This is the format of
       Word, Excel and PowerPoint files written by Microsoft Office 2007 --
       essentially ZIP archives of XML files.
