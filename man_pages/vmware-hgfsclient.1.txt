NAME

vmware&minus;hgfsclient
&minus; hgfs client access

SYNOPSIS

vmware&minus;hgfsclient

DESCRIPTION

This is a
simple console application and can be run as any user.
vmware&minus;hgfsclient is a userspace HGFS client
implementation. It will probably one day become a full
featured HGFS filesystem driver.

OPTIONS

vmware&minus;hgfsclient
has no options.
