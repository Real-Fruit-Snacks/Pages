NAME
       Image::ExifTool::DJI - DJI Phantom maker notes tags

SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.

DESCRIPTION
       This module contains definitions required by Image::ExifTool to
       interpret the maker notes in images from some DJI Phantom drones.
