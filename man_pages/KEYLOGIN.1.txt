NAME

keylogin &mdash; decrypt
and store secret key

SYNOPSIS

keylogin

DESCRIPTION

The keylogin utility
prompts the user for their login password, and uses it to
decrypt the user&rsquo;s secret key stored in the
publickey(5) database. Once decrypted, the user&rsquo;s key
is stored by the local key server process keyserv(8) to be
used by any secure network services, such as NFS.
