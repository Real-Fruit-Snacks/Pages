NAME
       mate-notification-properties - Set up the options for desktop notifica-
       tions

SYNOPSIS
       mate-notification-properties

DESCRIPTION
       This manual page  documents  briefly  the  mate-notification-properties
       command.

       mate-notification-properties is a utility to set up various options re-
       lated to desktop notifications.
