NAME
       Text::Xslate::PP::Opcode - Text::Xslate opcode implementation in pure
       Perl

DESCRIPTION
       This module is a pure Perl implementation of the Xslate opcodes.

       The is enabled with "$ENV{ENV}='pp=opcode'".
