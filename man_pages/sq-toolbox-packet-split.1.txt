NAME

sq toolbox
packet split - Split a message into packets

SYNOPSIS

sq toolbox
packet split [OPTIONS] FILE

DESCRIPTION

Split a message
into packets.

Splitting a
packet sequence into individual packets, then recombining
them freely with &rsquo;sq toolbox packet join&rsquo; is a
great way to experiment with OpenPGP data.

The converse
operation is &rsquo;sq toolbox packet join&rsquo;.

OPTIONS

Subcommand
options 
-B, --binary

Emit binary data

-p,
--prefix=PREFIX

Write to files with PREFIX
[defaults: &rsquo;FILE-&rsquo; if FILE is set, or
&rsquo;output-&rsquo; if read from stdin]

FILE

Read from FILE or stdin if omitted

Global
options 
See sq(1) for a description of the global
options.

EXAMPLES

Split a
certificate into individual packets

sq toolbox
packet split juliet.pgp

SEE ALSO

sq(1),
sq-toolbox(1), sq-toolbox-packet(1).

For the full
documentation see https://book.sequoia-pgp.org&gt;.
