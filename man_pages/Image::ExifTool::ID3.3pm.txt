NAME
       Image::ExifTool::ID3 - Read ID3 meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       ID3 information from audio files.  ID3 information is found in MP3 and
       various other types of audio files.
