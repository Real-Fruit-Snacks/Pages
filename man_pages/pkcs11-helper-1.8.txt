NAME
     pkcs11-helper — Simplified PKCS#11 library.

DESCRIPTION
     pkcs11-helper is a simplified API for accessing PKCS#11 tokens for user
     applications.

SEE ALSO
     pkcs11-helper Home Page, https://github.com/OpenSC/pkcs11-helper/.
