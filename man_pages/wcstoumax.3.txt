NAME

wcstoimax,
wcstoumax - convert wide-character string to integer

SYNOPSIS

#include

#include 

intmax_t
wcstoimax(const wchar_t *nptr, wchar_t
**endptr, int base); 
uintmax_t wcstoumax(const wchar_t *nptr,
wchar_t **endptr, int
base);

DESCRIPTION

These functions
are just like wcstol(3) and wcstoul(3), except
that they return a value of type intmax_t and
uintmax_t, respectively.
