НАЗВАНИЕ
       debconf-show - выполнить запрос к базе данных debconf

СИНТАКСИС
        debconf-show имя-пакета [...] [--db=имя-базы-данных]
        debconf-show --listowners [--db=имя-базы-данных]
        debconf-show --listdbs

ОПИСАНИЕ
       debconf-show позволяет выполнить запрос к базе данных debconf
       различными способами.

       Наиболее распространённый запрос — «debconf-show имя-пакета», по
       которому выдаются все элементы базы данных debconf принадлежащие этому
       пакету и их текущие значения. Вопросы, которые были уже заданы,
       отмечены символом «*».

       Эта программа может быть полезна при отладке и особенно при отправке
       отчётов об ошибке в пакетах использующих debconf.

ПАРАМЕТРЫ
       --db=имя-базы-данных
           Указать базу данных для запроса. По умолчанию debconf-show
           выполняет запрос к главной базе данных.

       --listowners
           Запросить список всех владельцев вопросов в базе данных. В
           большинстве случаев владелец — это имя пакета debian.

       --listdbs
           Показать список доступных баз данных.
