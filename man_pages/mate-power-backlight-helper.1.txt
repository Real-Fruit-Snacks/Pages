DESCRIPTION

mate-power-backlight-helper
is a helper utility for controlling the backlight of TFT
displays via the MATE power manager userspace daemon.

The
mate-power-backlight-helper requires to be run with
root privileges.

OPTIONS

--help

Show summary of options.

--set-brightness
BRIGHTNESS_LEVEL

Set the given brightness.

--get-brightness

Get the current brightness.

--get-max-brightness

Get the number of brightness
levels supported.
