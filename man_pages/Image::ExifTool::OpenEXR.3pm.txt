NAME
       Image::ExifTool::OpenEXR - Read OpenEXR meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information from OpenEXR images.
