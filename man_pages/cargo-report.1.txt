NAME

cargo-report
&mdash; Generate and display various kinds of reports

SYNOPSIS

cargo
report type [options]

DESCRIPTION

Displays a report of the given type &mdash;
currently, only future-incompat is supported

OPTIONS

--id
id

Show the report with the
specified Cargo-generated id

-p
spec&#8230;, --package spec&#8230;

Only display a report for the
specified package

EXAMPLES

1. Display the
latest future-incompat report:

cargo report
future-incompat

2. Display the
latest future-incompat report for a specific package:

cargo report
future-incompat --package my-dep:0.0.1
