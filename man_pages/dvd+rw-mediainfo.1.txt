NAME
       dvd+rw-mediainfo - display information about dvd drive and disk

SYNOPSIS
       dvd+rw-mediainfo /dev/dvd

DESCRIPTION
       The  dvd+rw-mediainfo  command displays information about the drive and
       disk in the specified dvd drive (e.g.  /dev/dvd).  This  is  especially
       useful  for  debugging  purposes and it is requested that its output be
       included when reporting bugs found in the dvd+rw-tools package.

OPTIONS
       This program does not accept any command line options.
