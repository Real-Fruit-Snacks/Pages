NAME

sasl_chalprompt_t
- Cyrus SASL documentation

SYNOPSIS

#include

int
sasl_chalprompt_t(void *context, int id, 
const char *challenge, 
const char *prompt, const char *defresult, 
const char **result, unsigned *len)

DESCRIPTION

int
sasl_chalprompt_t(void *context, 
int id, 
const char *challenge, 
const char *prompt, 
const char *defresult, 
const char **result, 
unsigned *len)

sasl_chalprompt_t is
used to prompt for input in response to a server challenge.

Parameters

&bull;

context &ndash; is the context from the callback
record 

&bull;

id &ndash; is the callback id (either
SASL_CB_ECHOPROMPT or SASL_CB_NOECHOPROMPT)

&bull;

challenge &ndash; the server&rsquo;s
challenge 

&bull;

prompt &ndash; A prompt for the user

&bull;

defresult &ndash; Default result (may be
NULL) 

&bull;

len &ndash; Length of the user&rsquo;s
response. 

Result

The
user&rsquo;s response (a NUL terminated string) or SASL
error code.

RETURN VALUE

The
user&rsquo;s response (NUL terminated), or a SASL error
code. See sasl_errors(3).

SEE ALSO

sasl(3),
sasl_errors(3), sasl_callbacks(3)
