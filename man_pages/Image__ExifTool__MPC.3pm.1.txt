NAME
       Image::ExifTool::MPC - Read Musepack audio meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains definitions required by Image::ExifTool to extract
       meta information from Musepack (MPC) audio files.
