NAME

range - index
range for a subset of a vector (rheolef-7.2)

DESCRIPTION

This class
represents a range of index for addressing a subset of the
vec(4) container.

EXAMPLE

vec
x (100, 3.14); 
vec y = x(range(0,50)); 
dout 

IMPLEMENTATION

This
documentation has been generated from file
linalg/lib/range.h

struct range {

using size_type = size_t; 
range (size_type start=0, size_type stop=0); 
size_type start() const; 
size_type size() const; 
static range all();

};
