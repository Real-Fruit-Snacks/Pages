NAME

gh-project-item-archive
- Archive an item in a project

SYNOPSIS

gh project
item-archive [] [flags]

OPTIONS

--format

Output format: {json}

--id 

ID of the item to archive

-q, --jq

Filter JSON output using a jq
expression

--owner

Login of the owner. Use
"@me" for the current user.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

--undo

Unarchive an item

EXAMPLE

# archive an
item in the current user&rsquo;s project "1" 
gh project item-archive 1 --owner "@me" --id
