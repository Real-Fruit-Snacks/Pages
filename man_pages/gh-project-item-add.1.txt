NAME

gh-project-item-add
- Add a pull request or an issue to a project

SYNOPSIS

gh project
item-add [] [flags]

OPTIONS

--format

Output format: {json}

-q, --jq

Filter JSON output using a jq
expression

--owner

Login of the owner. Use
"@me" for the current user.

-t, --template

Format JSON output using a Go
template; see "gh help formatting"

--url 

URL of the issue or pull
request to add to the project

EXAMPLE

# add an item
to monalisa&rsquo;s project "1" 
gh project item-add 1 --owner monalisa --url
https://github.com/monalisa/myproject/issues/23
