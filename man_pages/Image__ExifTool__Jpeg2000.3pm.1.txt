NAME
       Image::ExifTool::Jpeg2000 - Read JPEG 2000 meta information

SYNOPSIS
       This module is used by Image::ExifTool

DESCRIPTION
       This module contains routines required by Image::ExifTool to read JPEG
       2000 files.
