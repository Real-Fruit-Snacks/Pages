NAME

pmNewZone
- establish a reporting timezone

C SYNOPSIS

#include

int
pmNewZone(const char *tz);

cc ...
-lpcp

DESCRIPTION

The current
reporting timezone affects the timezone used by
pmCtime(3) and pmLocaltime(3).

The argument
tz defines a timezone string, in the format described
for the TZ environment variable, see
environ(7).

pmNewZone
sets the current reporting timezone, and returns a value
that may be used in a subsequent call to pmUseZone(3)
to restore this reporting timezone.

DIAGNOSTICS

A return value
less than zero indicates a fatal error from a system call,
most likely malloc(3).
